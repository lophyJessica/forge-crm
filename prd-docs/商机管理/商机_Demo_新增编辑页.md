# 商机_Demo_新增编辑页

> **版本**：V1.0 | 2026-07-17

---

## 1. 页面概述

| 项目 | 内容 |
|------|------|
| 新增路径 | `/opportunities/new` |
| 编辑路径 | `/opportunities/{id}/edit` |
| 骨架结构 | 表单卡片 + 固定底部操作栏 |

---

## 2. 底部操作栏

| 按钮 | 样式 | 说明 |
|------|------|------|
| 返回列表 | 次要 | 不保存 |
| 保存 | 主按钮 | 保存后停留在编辑页 |

---

## 3. 表单字段

| 字段 | 控件 | 规格 |
|------|------|------|
| 商机名称 | Input | 必填,上限100字符 |
| 关联客户 | SelectWithSearch | 必填,从CRM客户列表搜索 |
| 预计金额 | NumberInput | 选填,前缀¥,保留2位小数 |
| 预计成交日期 | DatePicker | 选填 |
| 需求描述 | Textarea | 选填,上限1000字符 |
| 关联商品 | MultiSelect | 选填(创建时),从ERP商品列表选择 |

---

## 4. 编辑模式

仅 INITIAL_CONTACT 和 NEEDS_CONFIRM 阶段可编辑。PROPOSAL 及以上只读。

---
