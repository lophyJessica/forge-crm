# 商机_Demo_详情页

> **版本**：V1.0 | 2026-07-17

---

## 1. 页面概述

| 项目 | 内容 |
|------|------|
| 页面路径 | `/opportunities/{id}` |
| 骨架结构 | 状态区 → 阶段时间线 → 信息卡片 → 商品明细 → 跟进记录 → 底部操作 |

---

## 2. 状态区

Badge 当前阶段 + AI 成交概率

## 3. 阶段时间线

7 个阶段节点,已完成的高亮,当前阶段的闪烁,未完成灰色。格式:`阶段名 - 到达时间`

## 4. 信息卡片

Key-Value 只读:商机名称、关联客户(可点击跳转)、预计金额、预计成交日期、需求描述、合同编号(CONTRACT/WON 阶段显示)

## 5. 商品明细卡片

表格:商品编码、商品名称、单价、数量。仅 NEEDS_CONFIRM 及以上展示。

## 6. 跟进记录

时间线,关联商机ID,同线索模块跟进记录组件。

## 7. 底部操作

| 阶段 | 按钮 |
|------|------|
| INITIAL/NEEDS/PROPOSAL | 返回/编辑/推进/输单 |
| NEGOT | 返回/推进/输单 |
| CONTRACT | 返回/查看合同/赢单 |
| WON/LOST | 返回 |
