> **版本**：V1.0 | 2026-07-17

## 1. 页面概述

| 项目 | 内容 |
|------|------|
| 页面名称 | 商机看板 |
| 页面目标 | 以看板视图展示商机 pipeline,支持拖拽推进,也可切换列表视图 |
| 页面路径 | `/opportunities` |
| 骨架结构 | 工具栏 + 看板视图(默认) / 列表视图(切换) |

## 2. 工具条

| 按钮 | 位置 | 说明 |
|------|------|------|
| 新建商机 | 左侧 | 跳转新增页 |
| 看板/列表切换 | 右侧 | 视图切换,默认看板 |

---

## 3. 看板视图

### 列定义(按阶段)

| 列名 | 阶段 | 颜色 |
|------|------|------|
| 初步接触 | INITIAL_CONTACT | 灰色 |
| 需求确认 | NEEDS_CONFIRM | 蓝色 |
| 方案报价 | PROPOSAL | 橙色 |
| 商务谈判 | NEGOTIATION | 黄色 |
| 合同签订 | CONTRACT | 紫色 |
| 赢单 | WON | 绿色 |
| 输单 | LOST | 红色 |

### 卡片内容

每张卡片展示:商机名称、客户名、预计金额(¥)、AI 成交概率(Tag)、预计成交日期

### 拖拽交互
- 仅可拖到右侧相邻列(逐级推进)
- 拖拽到非相邻列 → 阻断"阶段必须逐级推进"
- 拖拽到输单列 → 弹出输单原因填写 Modal

## 4. 列表视图

### Tab

| Tab | 过滤 |
|-----|------|
| 全部 | 所有阶段 |
| 进行中 | INITIAL+NEEDS+PROPOSAL+NEGOT+CONTRACT |
| 赢单 | WON |
| 输单 | LOST |

### 列定义

| 列名 | 宽度 |
|------|------|
| 商机编号 | 160px |
| 商机名称 | 200px |
| 客户名 | 160px |
| 预计金额 | 140px(¥千分位) |
| 商机阶段 | 120px |
| AI 成交概率 | 100px(≥70绿/40-69黄/<40红) |
| 预计成交日期 | 140px |
| 创建时间 | 160px |
| 操作 | 120px |

### 行内操作

| 阶段 | 操作 |
|------|------|
| INITIAL/NEEDS/PROPOSAL | 查看/编辑/推进/输单 |
| NEGOT | 查看/推进/输单 |
| CONTRACT | 查看合同/赢单 |
| WON/LOST | 查看 |
