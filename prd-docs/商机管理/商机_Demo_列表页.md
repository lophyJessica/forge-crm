# 商机_Demo_列表页

> **版本**：V2.0 | 2026-07-18
> **页面类型**：PC Web 列表页 / 看板页
> **前置文档**：《商机主PRD》《商机字段清单》
> **字段约束**：字段取值、必填性、格式和 Tag 配色全部引用《商机字段清单》，本文不重复定义。

---

## 1. 页面概述

| 项目 | 内容 |
|------|------|
| 页面名称 | 商机管理 |
| 页面目标 | 在看板和列表两种视图下查看、筛选、推进与关闭商机 |
| 页面路径 | `/opportunities` |
| 默认视图 | 看板视图 |
| 默认排序 | 引用《商机字段清单》展示规则 |
| 页面权限 | 服从《商机主PRD》数据范围与操作权限矩阵 |
| 页面宽度 | 主内容区最小宽度 1200px，低于时出现横向滚动 |
| 加载方式 | 首屏骨架屏，查询完成后替换为真实内容 |

页面由以下区域自上而下组成：

1. 标题与主操作区。
2. 状态 Tab 栏。
3. 查询区。
4. 工具条与视图切换。
5. 看板或表格内容区。
6. 分页区。

## 2. 状态 Tab 与后台计数

### 2.1 Tab 规格

| Tab | 过滤语义 | 计数来源 | 点击行为 |
|-----|----------|----------|----------|
| 全部 | 当前数据权限范围内全部商机 | 后台聚合接口 | 清空阶段聚合条件，保留其他查询条件 |
| 进行中 | 所有非终态商机 | 后台聚合接口 | 刷新列表或看板并回到第一页 |
| 赢单 | 赢单终态商机 | 后台聚合接口 | 刷新内容并回到第一页 |
| 输单 | 输单终态商机 | 后台聚合接口 | 刷新内容并回到第一页 |

### 2.2 计数规则

- Tab 文案格式为 `Tab名称 (计数)`。
- 计数必须由后台按当前用户数据权限聚合，不使用当前页数据推算。
- 计数不受分页影响。
- 计数受查询区条件影响。
- 点击“重置”后重新请求全部计数。
- 状态推进成功后立即刷新受影响 Tab 的计数。
- 输单成功后“进行中”减一，“输单”加一。
- 合同签署联动赢单后“进行中”减一，“赢单”加一。
- 创建成功后“全部”和“进行中”各加一。
- 计数接口失败时显示 `--`，不阻断列表查询。
- 计数加载期间使用固定宽度骨架，避免 Tab 抖动。
- 快速切换 Tab 时取消上一次未完成请求。

### 2.3 Tab 状态保持

- URL 查询参数保存当前 Tab。
- 从详情页返回时恢复原 Tab、查询条件、页码和滚动位置。
- 直接访问无效 Tab 参数时回退到“全部”。
- 看板和列表切换时保留当前 Tab。

## 3. 查询区

### 3.1 布局

- 查询区默认单行展示，空间不足时换为两行。
- 第一行最多放置 5 个控件和查询按钮组。
- 控件标签在上、控件在下，便于长文本对齐。
- “查询”为主按钮，“重置”为次按钮。
- 回车仅在关键词输入框内触发查询。
- 查询触发后页码重置为 1。
- 查询参数同步写入 URL。

### 3.2 控件详细规格

| 字段名 | 控件 | 宽度 | Placeholder | 输入/选择规则 | 查询行为 |
|--------|------|------|-------------|---------------|----------|
| 商机名称 | Input | 220px | `请输入商机名称` | 支持清空；前后空格提交时去除 | 模糊匹配 |
| 商机编号 | Input | 200px | `请输入商机编号` | 支持粘贴；不自动补全 | 精确匹配优先 |
| 关联客户 | SelectWithSearch | 220px | `请选择关联客户` | 远程搜索；选项只显示可见客户 | 按客户标识过滤 |
| 商机阶段 | Select | 180px | `请选择商机阶段` | 选项引用字段清单；支持清空 | 与 Tab 条件取交集 |
| AI 成交概率 | RangeInput | 200px | `最小值 - 最大值` | 仅允许合法区间；最小值不得大于最大值 | 闭区间过滤 |
| 预计成交日期 | DateRangePicker | 260px | `开始日期 - 结束日期` | 支持清空；开始不得晚于结束 | 含首尾日期 |
| 创建人 | SelectWithSearch | 180px | `请选择创建人` | 仅主管和管理员展示；远程搜索 | 按用户标识过滤 |
| 创建时间 | DateRangePicker | 260px | `开始日期 - 结束日期` | 默认空；最大跨度 1 年 | 含首尾日期 |

### 3.3 查询反馈

- 查询中禁用“查询”按钮并显示加载图标。
- 查询成功后更新 Tab 计数和内容区。
- 无结果时显示 `未找到符合条件的商机`。
- 查询失败时保留原结果并 Toast `查询失败，请重试`。
- 重置清空所有查询条件，但保留当前视图模式。
- 重置后回到“全部”Tab 和第一页。

## 4. 工具条

| 控件 | 位置 | 样式 | 权限 | 行为 |
|------|------|------|------|------|
| 新建商机 | 左侧 | 主按钮，Plus 图标 | 有新增权限时展示 | 跳转新增页 |
| 刷新 | 左侧 | 图标按钮，RefreshCw 图标 | 全部角色 | 保留条件重新查询 |
| 看板/列表 | 右侧 | SegmentedControl | 全部角色 | 切换视图并写入本地偏好 |

- 导出进行中禁止重复点击。
- 导出完成 Toast `导出任务已创建`。
- 视图偏好保存在浏览器本地。

## 5. 看板视图

### 5.1 看板列

- 每个业务阶段对应一列。
- 列顺序引用《商机字段清单》展示规则。
- 列标题展示阶段名称、后台计数和金额汇总。
- 列宽固定 288px。
- 看板区域横向滚动，不压缩列宽。
- 每列独立按阶段变更时间倒序。
- 每列首批加载 20 张卡片。
- 列底部“加载更多”按该阶段分页。
- 终态列可查看但不可作为普通推进拖拽目标。

### 5.2 卡片结构

| 区域 | 内容 | 展示规则 |
|------|------|----------|
| 顶部 | 商机名称 | 两行截断，悬浮显示全文 |
| 标识 | 商机编号 | 点击进入详情 |
| 客户 | 关联客户 | 单行截断 |
| 金额 | 预计金额 | 右对齐，格式化展示 |
| 概率 | AI 成交概率 | 样式引用字段清单 |
| 日期 | 预计成交日期 | 无值显示 `--` |
| 底部 | 最近阶段时间 | 相对时间 + Tooltip 完整时间 |

卡片稳定尺寸：

- 最小高度 184px。
- 卡片圆角 6px。
- 内边距 12px。
- 卡片之间间距 10px。
- 拖拽占位符与卡片同高。
- 加载概率时不改变卡片高度。

### 5.3 拖拽详细规则

1. 仅有推进权限且商机非终态时允许抓取。
2. 抓取区域为整张卡片，链接点击不触发拖拽。
3. 开始拖拽时原卡片保留半透明占位。
4. 合法目标列高亮边框并显示 `松开以推进`。
5. 仅相邻下一阶段为普通合法目标。
6. 向左拖拽一律阻断。
7. 跨越一个或多个阶段一律阻断。
8. 拖到赢单列一律阻断，赢单只接受合同签署事件。
9. 拖到输单列不直接提交，先打开输单弹窗。
10. 拖到合同阶段时打开“发起合同”确认弹窗。
11. 推进前执行与主 PRD 相同的前置校验。
12. 校验失败时卡片回到原位置。
13. 请求期间锁定卡片，显示局部加载态。
14. 请求成功后刷新来源列、目标列和 Tab 计数。
15. 请求失败时回滚卡片并 Toast 具体原因。
16. 并发冲突时刷新卡片最新状态并提示 `商机阶段已变化`。
17. 键盘用户可通过卡片菜单执行等价“推进”动作。
18. 拖拽行为写入阶段变更审计记录。

### 5.4 看板空态

- 单列为空时显示 `该阶段暂无商机`。
- 全部列为空时显示全局空状态。
- 有新增权限时全局空状态提供“新建商机”。
- 查询导致空态时提供“重置筛选”。

| 商机阶段 | 主操作 | 次操作 | More 菜单 |

- 不可用动作不渲染 disabled 状态。

- 操作成功后只刷新当前行和后台计数。

### 7.1 推进阶段

| 项目 | 文案/行为 |
|------|-----------|
| 标题 | `推进商机` |
| 内容 | `确认将商机“{商机名称}”推进至“{目标阶段}”吗？` |

| 项目 | 文案/行为 |
|------|-----------|
| 标题 | `发起合同` |

### 7.3 输单关闭

| 项目 | 文案/行为 |
|------|-----------|
| 标题 | `确认输单` |
| 内容 | `输单后商机将关闭且不可恢复，请填写输单原因。` |
| 输入字段 | 输单原因；约束引用字段清单 |

弹窗通用行为：

- 打开后焦点进入第一个可交互控件。
- 点击遮罩不关闭危险操作弹窗。
- 提交中禁用全部按钮。
- Esc 仅在未提交时关闭。
- 服务端失败保留用户已填内容。

## 8. 分页与刷新

- 默认每页 20 条。
- 支持 20、50、100 条切换。
- 分页显示总条数、页码和每页数量。
- 切页后滚动到表格顶部。
- 看板按列独立分页，不展示底部分页器。
- 自动刷新间隔为 60 秒，仅页面可见时运行。
- 用户正在拖拽或弹窗编辑时暂停自动刷新。
- 自动刷新不覆盖用户当前输入。
