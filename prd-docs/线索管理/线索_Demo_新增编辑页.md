# 线索_Demo_新增编辑页

> **文档定位**：AI 生成线索新增编辑页前端原型的输入文档
> **字段定义**：字段枚举值、必填性等见《线索字段清单》，本文件只写页面呈现规则，严禁重复定义
> **版本**：V1.0 | 2026-07-17

---

## 1. 页面概述

| 项目 | 内容 |
|------|------|
| 页面名称 | 线索新增 / 编辑 |
| 页面目标 | 创建新线索或编辑草稿态线索 |
| 新增路径 | `/leads/new` |
| 编辑路径 | `/leads/{id}/edit` |
| 骨架结构 | 表单卡片 + 固定底部操作栏 |

---

## 2. 固定底部操作栏

| 元素 | 规格 |
|------|------|
| 位置 | `fixed bottom-0 right-0`，宽度与表单卡片一致 |
| 按钮顺序 | `返回列表` → `保存草稿` → `提交` |
| `保存草稿` | 次要按钮，保存后状态为 DRAFT |
| `提交` | 主按钮（实心），保存后状态为 PENDING_ASSIGN 并触发 AI 评分 |

---

## 3. 表单卡片

### 3.1 基本信息

卡片标题：**基本信息**。栅格：4 列桌面端。

| 字段 | 控件类型 | 详细规格 |
|------|---------|---------|
| 线索来源 | Select | 下拉选择：官网/线下活动/展会/转介绍/批量导入/其他 |
| 公司名称 | Input | placeholder：请输入公司名称，上限 100 字符 |
| 联系人 | Input | placeholder：请输入联系人姓名，上限 50 字符 |
| 手机号 | Input | placeholder：请输入 11 位手机号，唯一校验 |
| 邮箱 | Input | placeholder：请输入邮箱，唯一校验 |
| 职位 | Input | placeholder：请输入职位，上限 50 字符 |
| 所属行业 | Select | 下拉选择：制造业/零售/医疗/金融/信息技术/其他 |
| 所在地区 | Cascader | 省-市级联选择，placeholder：请选择所在地区 |
| 线索备注 | Textarea | placeholder：请输入备注信息（选填），上限 500 字符 |

**手机号与邮箱校验规则**：至少填写一个，为空时提示"手机号和邮箱至少填写一个"

---

## 4. 编辑模式差异

| 字段 | DRAFT 草稿态 | 其他状态 |
|------|:-----------:|:------:|
| 全部字段 | 可编辑 | 只读（编辑仅限 DRAFT 态） |

---

## 5. 交互规范

### 5.1 保存草稿
- 不校验必填字段完整度
- 状态 = DRAFT
- API 调用后提示"草稿已保存"

### 5.2 提交
- 校验必填字段 + 手机号/邮箱去重
- 校验通过 → 状态 = PENDING_ASSIGN，触发 AI 评分
- 校验失败 → 标红缺失字段，Toast "请完善必填信息"
- 去重失败 → Toast "该手机号/邮箱已存在线索 XXX"

### 5.3 返回
- 点击「返回列表」→ 回到上一页，不保存
