| :--- | :--- | :--- |

## 2. 列表页

### 2.1 页面结构

| 区域 | 内容 | 说明 |
| :--- | :--- | :--- |
| 标题区 | 标题“库区管理”、说明文案、新增库区按钮 | 新增按钮进入新增库区页 |
| 查询区 | 综合搜索、所属仓库、库区类型、状态、查询按钮、重置按钮 | 类型和状态选项引用字段清单 |
| 表格区 | 库区列表、状态标识、行操作 | 默认分页 20 条；空数据展示“暂无库区档案” |

### 2.2 查询控件

| 控件 | 类型 | 交互 |
| :--- | :--- | :--- |
| 综合搜索 | Input | 支持库区编码 / 库区名称模糊查询；回车或点击查询触发 |
| 所属仓库 | Select | 选项来源于仓库档案；默认全部 |
| 库区类型 | Select | 默认全部；选项引用字段清单 |
| 状态 | Select | 默认全部；选项引用字段清单 |
| 查询 | Button | 触发筛选 |
| 重置 | Button | 清空条件并刷新列表 |

### 2.3 表格列与行操作

| 列 | 展示规则 |
| :--- | :--- |
| 库区编码 | 等宽文本，点击可进入详情页 |
| 库区名称 | 普通文本 |
| 所属仓库 | 展示仓库名称；空值展示 `-` |
| 库区类型 | 展示字段清单中的枚举文案 |
| 状态 | StatusPill，启用 / 停用使用不同颜色 |
| 最后修改时间 | 使用 `YYYY-MM-DD HH:mm:ss` |
| 操作 | 查看、编辑、停用 / 启用 |

行操作规则：

- 查看：进入详情页。
- 编辑：进入编辑库区页。
- 停用：仅启用行展示，点击后弹出确认框，确认后状态变为停用。
- 启用：仅停用行展示，点击后弹出确认框，确认后状态变为启用。
- 不提供删除入口。
- 按通用规范，当前状态下不可用的动作隐藏，不展示灰色 disabled 态。

## 3. 新增 / 编辑页

### 3.1 页面结构

| 区域 | 新增模式 | 编辑模式 |
| :--- | :--- | :--- |
| 顶部导航 | 返回按钮、标题“新增库区” | 返回按钮、标题“编辑库区 {code}” |
| 基础信息表单 | 库区编码、库区名称、所属仓库、库区类型、备注 | 库区编码只读，其余基础字段按字段清单维护 |
| 操作区 | 取消、保存 | 取消、保存 |

### 3.2 表单控件

| 字段 | 控件 | 新增模式 | 编辑模式 |
| :--- | :--- | :--- | :--- |
| 库区编码 | Input | 可编辑 | 只读 |
| 库区名称 | Input | 可编辑 | 可编辑 |
| 所属仓库 | Select | 可编辑 | 可编辑 |
| 库区类型 | Select | 可编辑 | 可编辑 |
| 备注 | Textarea | 可编辑 | 可编辑 |

状态字段不在新增 / 编辑主表单中展示；状态只能在列表页或详情页通过启用 / 停用按钮变更。

### 3.3 表单交互

| 场景 | 交互 |
| :--- | :--- |
| 所属仓库加载 | 从仓库档案读取启用仓库作为候选，不出现手工新增仓库入口 |
| 保存成功 | Toast 提示“库区档案已保存”；返回列表页并刷新数据 |
| 新增成功 | Toast 提示“库区档案已新增”；返回列表页并展示新记录 |
| 保存失败 | 阻断提交，定位首个错误字段或展示错误提示 |
| 取消 | 返回列表页，不保存当前输入 |
| 编辑不存在编码 | Toast / Alert 提示“库区档案不存在”，返回列表页 |

保存前按《库区字段清单》执行必填、格式、唯一性、枚举、所属仓库有效性校验。

## 4. 详情页

### 4.1 页面结构

| 区域 | 内容 | 说明 |
| :--- | :--- | :--- |
| 顶部区 | 返回按钮、标题、状态标识、编辑按钮、启用 / 停用按钮 | 启停用按钮按当前状态切换展示 |
| 基础信息 | 库区编码、库区名称、所属仓库、库区类型、备注 | 只读展示，空值展示 `-` |
| 系统信息 | 创建时间、最后修改时间 | 使用 `YYYY-MM-DD HH:mm:ss` |
| 关联摘要 | 关联货位数 | 从 Demo 数据按库区编码聚合；无数据展示 0 |

### 4.2 详情页交互

- 点击编辑进入编辑库区页。
- 点击停用 / 启用弹出二次确认，确认后刷新详情页状态。
- 停用后详情页仍可查看，关联货位摘要不隐藏。
- 不提供删除入口。
- 若记录不存在，提示“库区档案不存在”并返回列表页。

## 6. 验收清单

| 验收ID | 验收项 | 标准 |
| :--- | :--- | :--- |
| FE-AC01 | 列表查询 | 综合搜索、所属仓库、类型筛选、状态筛选可组合使用，重置后恢复全部数据 |
| FE-AC02 | 新增保存 | 必填字段填写完整后可新增，保存后返回列表并看到新库区 |
| FE-AC03 | 仓库来源 | 所属仓库下拉只来自仓库档案，不出现手工新增仓库入口 |
| FE-AC04 | 编辑保存 | 编辑页库区编码只读，保存后列表和详情展示更新结果 |
| FE-AC05 | 状态变更 | 启用 / 停用通过按钮触发并二次确认，确认后列表与详情状态同步刷新 |
| FE-AC06 | 删除禁用 | 页面不出现删除按钮、删除菜单或物理删除交互 |
| FE-AC07 | 详情展示 | 详情页可展示基础信息、系统信息、关联货位数 |
| FE-AC08 | 字段口径 | 页面枚举、必填和校验口径不在本文硬编码解释，均引用字段清单 |
