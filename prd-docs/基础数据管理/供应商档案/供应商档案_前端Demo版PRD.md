# 供应商档案_前端Demo版PRD

> 版本：V1.0 | 更新时间：2026-07-07
> 读者：前端 Demo 实现、测试、产品复核
> 文档定位：定义供应商档案 Demo 的列表页、同步新增页、编辑页、详情页页面结构、控件、交互与 Mock 数据。字段定义、枚举、必填性和校验规则统一引用《供应商档案字段清单》，本文不重复定义。

## 1. 页面清单

| 页面 | 建议路由 | 页面目标 |
| :--- | :--- | :--- |
| 供应商列表页 | `/base/suppliers` | 查询供应商、查看状态、进入同步新增 / 编辑 / 详情、触发启停用 |
| 同步新增供应商页 | `/base/suppliers/new` | Demo 中模拟从 ERP 同步新增供应商，不代表生产 WMS 人工新增 ERP 主数据 |

| 供应商详情页 | `/base/suppliers/:code` | 只读查看供应商资料、同步信息与 WMS 引用说明 |

## 2. 列表页

### 2.1 页面结构

| 区域 | 内容 | 说明 |
| :--- | :--- | :--- |
| 标题区 | 标题“供应商档案”、同步新增按钮 | 同步新增按钮进入 Demo 模拟 ERP 同步新增页 |
| 查询区 | 综合搜索、结算方式、状态、查询按钮、重置按钮 | 选项引用字段清单 |
| 表格区 | 供应商列表、状态标识、行操作 | 默认分页 20 条；空数据展示“暂无供应商档案” |

### 2.2 查询控件

| 控件 | 类型 | 交互 |
| :--- | :--- | :--- |
| 综合搜索 | Input | 支持供应商编码 / 供应商名称 / 供应商简称 / 联系人模糊查询；回车或点击查询触发 |
| 结算方式 | Select | 默认全部；选项引用字段清单 `SettlementMethod` |
| 状态 | Select | 默认全部；选项引用字段清单 `BaseDataStatus` |
| 查询 | Button | 触发筛选 |
| 重置 | Button | 清空条件并刷新列表 |

### 2.3 表格列与行操作

| 列 | 展示规则 |
| :--- | :--- |
| 供应商编码 | 等宽文本，点击可进入详情页 |
| 供应商名称 | 普通文本，长文本截断 |
| 供应商简称 | 无值展示 `-` |
| 联系人 | 普通文本 |
| 联系电话 | 普通文本 |
| 结算方式 | 展示枚举文案，文案来源字段清单 |
| 账期天数 | 数字展示；现付供应商展示 `0` |
| 状态 | StatusPill，启用 / 停用使用不同颜色 |
| 最后同步时间 | `YYYY-MM-DD HH:mm:ss` |
| 操作 | 查看、编辑、停用 / 启用 |

行操作规则：

- 查看：进入详情页。
- 编辑：进入编辑供应商页；ERP 同步字段只读。
- 停用：仅 `ENABLED` 行展示，点击后弹出确认框，填写停用原因并确认后状态变为 `DISABLED`。
- 启用：仅 `DISABLED` 行展示，点击后弹出确认框，确认后状态变为 `ENABLED`。
- 不提供删除入口。
- 按通用规范，当前状态下不可用的动作隐藏，不展示灰色 disabled 态。

## 3. 同步新增 / 编辑页

### 3.1 页面定位

供应商档案来源于 ERP，生产 WMS 不人工新增或编辑 ERP 供应商主数据。Demo 为了展示完整页面链路，提供“同步新增供应商页”用于选择一条预置 Mock ERP 数据并确认同步，不代表 WMS 维护供应商主数据。

### 3.2 页面结构

| 区域 | 同步新增模式 | 编辑模式 |
| :--- | :--- | :--- |
| 顶部导航 | 返回按钮、标题“同步新增供应商” | 返回按钮、标题“编辑供应商 {code}” |
| ERP 同步信息 | 供应商编码、名称、简称、联系人、电话、地址、结算方式、账期、税率 | 同字段只读展示 |
| WMS 控制信息 | 默认状态为启用；不展示状态下拉 | 当前状态只读展示；通过页面动作按钮启用 / 停用 |
| 操作区 | 取消、确认同步 | 返回、查看详情、启用 / 停用 |

### 3.3 表单控件

| 字段组 | 同步新增模式 | 编辑模式 |
| :--- | :--- | :--- |
| ERP 同步字段 | 从预置 Mock ERP 数据中选择后预览；保存文案为“确认同步” | 只读，不可编辑 |
| 状态 | 不展示状态下拉；确认同步后默认 `ENABLED` | 只读展示，不在主表单编辑 |
| 停用原因 | 不展示 | 仅在停用弹窗中展示并校验 |
| 系统字段 | 不展示 | 只读展示最后同步时间、创建时间、最后修改时间 |

### 3.4 保存与动作交互

| 场景 | 交互 |
| :--- | :--- |
| 确认同步成功 | Toast 提示“供应商已同步”；返回列表页并展示新记录 |
| 同步编码已存在 | 按字段清单规则更新同编码供应商资料，Toast 提示“供应商资料已更新” |
| 同步校验失败 | 阻断提交，定位首个错误字段或展示错误提示 |
| 编辑页返回 | 返回列表页，不保存 ERP 字段变更 |
| 停用成功 | Toast 提示“供应商已停用”；刷新列表或详情状态 |
| 启用成功 | Toast 提示“供应商已启用”；刷新列表或详情状态 |

## 4. 详情页

### 4.1 页面结构

| 区域 | 内容 | 说明 |
| :--- | :--- | :--- |
| 顶部区 | 返回按钮、标题、状态标识、编辑按钮、启用 / 停用按钮 | 启停用按钮按当前状态切换展示 |
| 基础识别 | 供应商编码、供应商名称、供应商简称 | ERP 同步字段只读 |
| 联系信息 | 联系人、联系电话、联系地址 | ERP 同步字段只读，空值展示 `-` |
| 经营信息 | 结算方式、账期天数、税率 | ERP 同步字段只读 |
| WMS 控制信息 | 状态、停用原因 | 状态按钮驱动；启用状态停用原因展示 `-` |
| 系统信息 | 来源系统、最后同步时间、创建时间、最后修改时间 | 时间格式引用字段清单 |
| 引用说明 | 收货单引用、退货处理引用 | 说明单据中供应商信息按快照存储 |

### 4.2 详情页交互

- 点击编辑进入编辑供应商页。
- 点击停用 / 启用弹出二次确认，确认后刷新详情页状态。
- 停用后详情页仍可查看，历史引用说明不隐藏。
- 不提供删除入口。
- ERP 同步字段不提供直接编辑入口。

## 6. 验收清单

| 验收ID | 验收项 | 标准 |
| :--- | :--- | :--- |
| FE-AC01 | 列表查询 | 综合搜索、结算方式筛选、状态筛选可组合使用，重置后恢复全部数据 |
| FE-AC02 | 同步新增 | Demo 可模拟从 ERP 同步新增供应商，保存后返回列表并展示新记录 |
| FE-AC03 | ERP 字段只读 | 编辑页、详情页中 ERP 同步字段均不可编辑，不出现保存 ERP 字段的入口 |
| FE-AC04 | 状态变更 | 启用 / 停用通过按钮触发并二次确认，停用时校验停用原因 |
| FE-AC05 | 删除禁用 | 页面不出现删除按钮、删除菜单或物理删除交互 |
| FE-AC06 | 详情展示 | 详情页展示基础识别、联系信息、经营信息、WMS 控制信息和系统信息 |
| FE-AC07 | 快照说明 | 页面说明供应商档案变更不影响历史收货单、退货记录中的供应商快照 |
| FE-AC08 | Mock 日期 | Mock 数据中的同步时间、创建时间、更新时间均使用 2026 年 |

## 7. 异常态与空状态

| 状态 | 展示 |
| :--- | :--- |
| 无供应商数据 | 列表空态展示“暂无供应商档案”，保留同步新增入口 |
| 查询无结果 | 展示“未找到符合条件的供应商”，保留重置按钮 |
| 供应商不存在 | 详情 / 编辑页提示“供应商档案不存在”，返回列表页 |
| 同步失败 | 展示错误提示，不写入 Dexie 数据 |
| 停用原因未填 | 停用弹窗内字段标红提示“停用原因不能为空” |

## 8. 不确定性说明

| 事项 | 说明 |
| :--- | :--- |
| 同步新增页 | 生产 WMS 不人工新增 ERP 供应商；本页仅为前端 Demo 模拟 ERP 同步预览。若后续接真实接口，应替换为同步任务或同步日志入口。 |
| ERP 状态同步 | 本 Demo 按 WMS 本地状态按钮控制候选范围；ERP 状态是否强制覆盖 WMS 状态需产品复核。 |
| 供应商简称 | 若真实 ERP 无简称字段，列表和详情展示 `-` 或隐藏该列，以字段清单最终复核为准。 |
