# 货位_前端Demo版PRD

> 角色：前端Demo版PRD | 类型：档案主数据
> 页面只描述结构、控件、交互和 2026 Mock；字段定义和枚举以 `货位字段清单.md` 为准，不在本文重复维护。

## 1. 页面定位

货位管理 Demo 用于 PC 端维护货位档案，支持仓库主管查看货位、创建货位、编辑基础信息、绑定或换绑货位条码、启用/停用货位，并为 PUT 上架和 PICK 拣货提供可扫码的基础数据。

页面不实现真实后端逻辑，不处理库存过账，不提供物理删除。

## 2. 页面入口

| 入口 | 说明 |
|:--|:--|
| 菜单 | 基础数据管理 / 货位管理 |
| 关联跳转 | 库区管理详情、库存查询、上架单详情、拣货单详情 |

## 3. 列表页

### 3.1 页面结构

```text
货位列表
├─ 顶部筛选区
│  ├─ 关键词：货位编码/货位名称/货位条码
│  ├─ 所属仓库、所属库区、货架编号、状态、是否可用
│  └─ 查询、重置
├─ 操作区
│  └─ 新增货位
├─ 表格区
│  ├─ 货位编码、货位名称、所属仓库、所属库区
│  ├─ 货架编号、层、列、货位条码
│  ├─ 状态、是否可用、更新时间
│  └─ 行操作：查看详情、编辑、绑定/换绑条码、启用/停用
└─ 分页
   └─ 默认 20 条/页
```

### 3.2 查询规则

- 关键词支持货位编码、货位名称、货位条码模糊查询。
- 所属库区选项来源于库区档案；先选仓库时，库区下拉只展示该仓库下的库区。
- 状态和是否可用只作为筛选条件，枚举口径引用 `货位字段清单.md`。
- 空值展示 `-`。

### 3.3 行操作规则

| 场景 | 行操作 |
|:--|:--|
| 货位启用 | 查看详情、编辑、换绑条码、停用 |
| 货位停用 | 查看详情、编辑、绑定/换绑条码、启用 |
| 未绑定条码 | 查看详情、编辑、绑定条码、停用 |
| 按钮不可用 | 直接隐藏，不展示 disabled 态 |

列表不展示删除按钮。

## 4. 新增/编辑页

### 4.1 页面结构

```text
新增/编辑货位
├─ 所属关系
│  ├─ 所属库区
│  └─ 所属仓库（随库区带出）
├─ 货位结构
│  ├─ 货位编码
│  ├─ 货架编号、层、列
│  └─ 货位名称预览
├─ 条码绑定
│  ├─ 货位条码输入框
│  └─ 扫码录入/清空输入
├─ 备注
└─ 底部动作栏
   ├─ 取消
   └─ 保存
```

### 4.2 控件说明

| 控件 | 说明 |
|:--|:--|
| 所属库区 | 必选，从库区档案选择，不允许手工输入 |
| 所属仓库 | 随所属库区带出，只读展示 |
| 货位编码 | 新增时录入；编辑时默认只读，避免影响历史引用 |
| 货架编号/层/列 | 必填，用于定位货位结构 |
| 货位名称预览 | 根据货架编号、层、列生成页面预览，不作为正式编码规则 |
| 货位条码 | 支持键盘输入或扫码录入，保存前校验格式和唯一性 |
| 保存 | 全量校验通过后返回列表或进入详情页 |

### 4.3 交互规则

| 场景 | 规则 |
|:--|:--|
| 选择所属库区 | 自动带出所属仓库，并刷新货位结构重复校验范围 |
| 修改货架/层/列 | 实时刷新货位名称预览 |
| 货位编码重复 | 阻断保存，提示 `货位编码已存在` |
| 货架/层/列重复 | 阻断保存，提示 `该库区下已存在相同货架/层/列货位` |
| 条码重复 | 阻断保存，提示 `货位条码已被绑定` |
| 备注超长 | 阻断保存，提示 `备注不能超过 200 字符` |
| 点击取消 | 返回列表，不保存当前输入 |

状态不在新增/编辑页直接修改。

## 5. 详情页

### 5.1 页面结构

```text
货位详情
├─ 顶部摘要
│  ├─ 货位编码、货位名称、状态、是否可用
│  └─ 编辑、绑定/换绑条码、启用/停用、返回列表
├─ 所属关系
│  └─ 所属仓库、所属库区、库区类型
├─ 货位结构
│  └─ 货架编号、层、列
├─ 条码信息
│  └─ 当前条码、绑定人、绑定时间
├─ 系统信息
│  └─ 创建人、创建时间、更新人、更新时间
└─ 操作记录
   └─ 新增、编辑、绑定条码、换绑条码、启用、停用
```

### 5.2 动作规则

| 动作 | 入口 | 说明 |
|:--|:--|:--|
| 编辑 | 详情页、列表行操作 | 进入编辑页，状态仍不可直接编辑 |
| 绑定条码 | 未绑定条码时展示 | 打开条码绑定弹窗 |
| 换绑条码 | 已绑定条码时展示 | 打开条码换绑弹窗，保存后记录原条码和新条码 |
| 停用 | 启用货位展示 | 二次确认后状态变更，不提供删除 |
| 启用 | 停用货位展示 | 校验所属库区和条码后状态变更 |

### 5.3 条码绑定弹窗

```text
绑定/换绑货位条码
├─ 当前货位：货位编码、货位名称、所属库区
├─ 当前条码
├─ 新条码输入框
├─ 扫码录入
└─ 取消、确认绑定
```

交互规则：

- 新条码支持扫码录入，也支持手工输入。
- 输入后实时校验格式；点击确认时校验唯一性。
- 换绑成功后刷新详情页条码信息，并新增操作记录。
- 条码绑定不改变货位状态；是否可用由字段清单中的计算规则决定。

## 7. 空状态与异常态

| 状态 | 展示 |
|:--|:--|
| 无数据 | `暂无货位` |
| 查询无结果 | `未找到符合条件的货位` |
| 库区为空 | `请先维护库区档案` |
| 保存失败 | 保留表单输入，提示 `货位保存失败，请重试` |
| 条码绑定失败 | 保留弹窗输入，提示 `条码绑定失败，请重试` |

## 8. 测试关注点

- 列表默认分页 20 条。
- 所属库区必须来自库区档案，所属仓库随库区带出。
- 新增和编辑页不允许直接修改状态。
- 货位编码、货架/层/列组合、货位条码重复时均阻断保存。
- 绑定和换绑条码成功后，详情页和列表同步刷新。
- 启用/停用只通过按钮触发，且不出现删除入口。
- 停用货位仍可查看详情和操作记录，但不可作为新作业可用货位。
