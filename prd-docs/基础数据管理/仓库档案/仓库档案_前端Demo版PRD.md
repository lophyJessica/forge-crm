| :--- | :--- | :--- |

## 2. 列表页

### 2.1 页面结构

| 区域 | 内容 | 说明 |
| :--- | :--- | :--- |
| 标题区 | 标题“仓库档案”、说明文案、新增仓库按钮 | 新增按钮进入新增仓库页 |
| 查询区 | 综合搜索、仓库类型、状态、查询按钮、重置按钮 | 类型和状态选项引用字段清单 |
| 表格区 | 仓库列表、状态标识、行操作 | 默认分页 20 条；空数据展示“暂无仓库档案” |

### 2.2 查询控件

| 控件 | 类型 | 交互 |
| :--- | :--- | :--- |
| 综合搜索 | Input | 支持仓库编码 / 仓库名称 / 负责人模糊查询；回车或点击查询触发 |
| 仓库类型 | Select | 默认全部；切换后刷新列表 |
| 状态 | Select | 默认全部；切换后刷新列表 |
| 查询 | Button | 触发筛选 |
| 重置 | Button | 清空条件并刷新列表 |

### 2.3 表格列与行操作

| 列 | 展示规则 |
| :--- | :--- |
| 仓库编码 | 等宽文本，点击可进入详情页 |
| 仓库名称 | 普通文本 |
| 仓库类型 | 展示枚举文案，文案来源字段清单 |
| 负责人 | 普通文本 |
| 仓库地址 | 长文本截断，悬浮展示完整地址 |
| 状态 | StatusPill，启用 / 停用使用不同颜色 |
| 操作 | 查看、编辑、停用 / 启用 |

行操作规则：

- 查看：进入详情页。
- 编辑：进入编辑仓库页。
- 停用：仅 `ENABLED` 行展示，点击后弹出确认框，确认后状态变为 `DISABLED`。
- 启用：仅 `DISABLED` 行展示，点击后弹出确认框，确认后状态变为 `ENABLED`。
- 不提供删除入口。
- 按通用规范，当前状态下不可用的动作隐藏，不展示灰色 disabled 态。

## 3. 新增 / 编辑页

### 3.1 页面结构

| 区域 | 新增模式 | 编辑模式 |
| :--- | :--- | :--- |
| 顶部导航 | 返回按钮、标题“新增仓库档案” | 返回按钮、标题“编辑仓库 {code}” |
| 基础信息表单 | 仓库编码、仓库名称、仓库类型、负责人、仓库地址、备注 | 仓库编码只读，其余基础字段按字段清单维护 |
| 操作区 | 取消、保存 | 取消、保存 |

### 3.2 表单控件

| 字段 | 控件 | 新增模式 | 编辑模式 |
| :--- | :--- | :--- | :--- |
| 仓库编码 | Input | 可编辑 | 只读 |
| 仓库名称 | Input | 可编辑 | 可编辑 |
| 仓库类型 | Select | 可编辑 | 可编辑 |
| 负责人 | Input | 可编辑 | 可编辑 |
| 仓库地址 | Input 或 Textarea | 可编辑 | 可编辑 |
| 备注 | Textarea | 可编辑 | 可编辑 |

状态字段不在新增 / 编辑主表单中展示；状态只能在列表页或详情页通过启用 / 停用按钮变更。

### 3.3 保存交互

| 场景 | 交互 |
| :--- | :--- |
| 保存成功 | Toast 提示“仓库档案已保存”；返回列表页并刷新数据 |
| 新增成功 | Toast 提示“仓库档案已新增”；返回列表页并展示新记录 |
| 保存失败 | 阻断提交，定位首个错误字段或展示错误提示 |
| 取消 | 返回列表页，不保存当前输入 |
| 编辑不存在编码 | Toast / Alert 提示“仓库档案不存在”，返回列表页 |

## 4. 详情页

### 4.1 页面结构

| 区域 | 内容 | 说明 |
| :--- | :--- | :--- |
| 顶部区 | 返回按钮、标题、状态标识、编辑按钮、启用 / 停用按钮 | 启停用按钮按当前状态切换展示 |
| 基础信息 | 仓库编码、仓库名称、仓库类型、负责人、仓库地址、备注 | 只读展示，空值展示 `-` |
| 系统信息 | 创建时间、最后修改时间 | 使用 `YYYY-MM-DD HH:mm:ss` |
| 关联摘要 | 关联库区数、关联货位数 | 从 Demo 数据按仓库编码聚合；无数据展示 0 |

### 4.2 详情页交互

- 点击编辑进入编辑仓库页。
- 点击停用 / 启用弹出二次确认，确认后刷新详情页状态。
- 停用后详情页仍可查看，关联库区和货位摘要不隐藏。
- 不提供删除入口。

## 6. 验收清单

| 验收ID | 验收项 | 标准 |
| :--- | :--- | :--- |
| FE-AC01 | 列表查询 | 综合搜索、类型筛选、状态筛选可组合使用，重置后恢复全部数据 |
| FE-AC02 | 新增保存 | 必填字段填写完整后可新增，保存后返回列表并看到新仓库 |
| FE-AC03 | 编辑保存 | 编辑页仓库编码只读，保存后列表和详情展示更新结果 |
| FE-AC04 | 状态变更 | 启用 / 停用通过按钮触发并二次确认，确认后列表与详情状态同步刷新 |
| FE-AC05 | 删除禁用 | 页面不出现删除按钮、删除菜单或物理删除交互 |
| FE-AC06 | 详情展示 | 详情页可展示基础信息、系统信息、关联库区数、关联货位数 |
| FE-AC07 | 字段口径 | 页面枚举、必填和校验口径不在本文硬编码解释，均引用字段清单 |
