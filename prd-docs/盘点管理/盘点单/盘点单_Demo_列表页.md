# 盘点单_Demo_列表页

> 角色：Demo·列表页 | 类型：执行作业单
> 页面只描述结构、控件、交互和 Mock；字段定义以 `盘点单字段清单.md` 为准，不重复维护字段枚举。

## 1. 页面定位

盘点单列表页用于 PC 端创建和跟踪 CK，覆盖草稿、盘点中、待审核、已调整的盘点单。列表页承担状态入口和查询入口，不承载 PDA 扫码实盘。

## 2. 页面入口

| 入口 | 说明 |
|:--|:--|
| 菜单 | 盘点管理 / 盘点单 |
| 关联跳转 | 库存查询、库存流水、盘点单详情 |

## 3. 页面结构

```text
盘点单列表
├─ 顶部操作区
│  └─ 新建盘点单
├─ 筛选区
│  ├─ 关键词：盘点单号/货位/商品/盘点人/审核人
│  ├─ 状态、盘点类型、仓库、是否有差异
│  ├─ 创建时间范围
│  └─ 查询、重置
├─ 表格区
│  ├─ 盘点单号、状态、盘点类型、范围、仓库
│  ├─ 盘点人、审核人、明细行数、差异行数、最大差异率、重盘次数
│  ├─ 创建时间、锁定时间、调整时间
│  └─ 行操作：查看详情、开始盘点、进入PDA、审核、查看FL
└─ 分页
   └─ 默认 20 条/页
```

## 4. 行操作规则

| CK 状态 | 行操作 |
|:--|:--|
| 草稿 | 查看详情、开始盘点 |
| 盘点中 | 查看详情、进入 PDA 作业、生成差异报告 |
| 待审核 | 查看详情、审核 |
| 已调整 | 查看详情、查看 FL |

> 按钮不可用时隐藏。列表页不展示删除入口，不展示独立差异调整单入口。

## 5. 查询规则

- 关键词支持盘点单号、货位编码、商品编码、盘点人、审核人模糊查询。
- 状态筛选仅使用 CK 自身状态：草稿、盘点中、待审核、已调整。
- 是否有差异根据 `difference_line_count > 0` 判断。
- 默认按创建时间倒序展示。
- PC 表格默认 20 条/页。

## 6. 新建盘点单入口

| 区域 | 字段/控件 | 说明 |
|:--|:--|:--|
| 基础信息 | 盘点类型、仓库、盘点范围 | 保存草稿前必填 |
| 人员信息 | 盘点人、审核人 | 审核人必须具备主管或审核权限 |
| 计划信息 | 计划开始时间、计划结束时间、备注 | 备注 ≤200 字符 |
| 动作 | 保存草稿、开始盘点 | 开始盘点会锁定范围并冻结货位 |

## 8. 空状态与异常态

| 状态 | 展示 |
|:--|:--|
| 无数据 | `暂无盘点单` |
| 查询无结果 | `未找到符合条件的盘点单` |
| 列表加载失败 | 保留筛选条件，提示 `盘点单列表加载失败，请重试` |
| 开始盘点失败 | 提示失败原因，如 `货位存在未完成出入库任务，暂不可冻结` |

## 9. 测试关注点

- 列表页可创建 CK 草稿，但不能删除 CK。
- 草稿点击开始盘点后进入盘点中，并触发范围冻结。
- 盘点中才能进入 PDA 作业或生成差异报告。
- 待审核才能显示审核入口。
- 已调整才能显示查看 FL。
