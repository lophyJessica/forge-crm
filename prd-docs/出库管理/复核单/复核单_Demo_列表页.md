# 复核单_Demo_列表页

> 角色：Demo·列表页 | 类型：执行作业单
> 页面只描述结构、控件、交互和 Mock；字段定义以 `复核单字段清单.md` 为准，不重复维护字段枚举。

## 1. 页面定位

复核单列表页用于 PC 端查看由拣货完成下推生成的 CHECK，跟踪待复核、复核中、已复核、已退回拣货任务和差异异常。列表页不提供新增入口，避免绕过 PICK。

## 2. 页面入口

| 入口 | 说明 |
|:--|:--|
| 菜单 | 出库管理 / 复核单 |
| 关联跳转 | 拣货单详情、包裹详情、出库管理概览 |
| 工作站入口 | 复核台登录后进入待复核任务列表 |

## 3. 页面结构

```text
复核单列表
├─ 顶部筛选区
│  ├─ 关键词：复核单号/拣货单号/波次单号/发货箱/SKU
│  ├─ 状态、仓库、复核员、复核端、是否异常
│  ├─ 创建时间范围
│  └─ 查询、重置
├─ 表格区
│  ├─ 复核单号、来源拣货单、来源波次、状态、仓库、发货箱
│  ├─ 应复核总数、实复核总数、差异总数、差异行数、进度
│  └─ 行操作：查看详情、开始复核、继续复核、查看包裹
└─ 分页
   └─ 默认 20 条/页
```

## 4. 行操作规则

| 场景 | 行操作 |
|:--|:--|
| 状态=待复核 | 查看详情、开始复核 |
| 状态=复核中 | 查看详情、继续复核 |
| 状态=已复核 | 查看详情、查看包裹（有 PKG 时） |
| 状态=已退回拣货 | 查看详情、查看来源拣货单 |
| 状态=已作废 | 查看详情 |
| 异常标记=true | 展示异常标识，可进入详情查看差异明细和退回原因 |

> 列表页不展示“新增复核单”。开始/继续复核只是进入复核执行页，不允许直接改状态。

## 5. 查询规则

- 关键词支持复核单号、来源拣货单号、来源波次单号、发货箱/包厢号、商品编码模糊查询。
- 状态筛选只使用 CHECK 自身状态。
- 是否异常根据差异行数、错品、串箱、退回记录等系统字段汇总。
- 创建时间按 CHECK 生成时间筛选，不按下游包裹时间筛选。

## 7. 空状态与异常态

| 状态 | 展示 |
|:--|:--|
| 无数据 | `暂无复核单` |
| 查询无结果 | `未找到符合条件的复核单` |
| 接口失败 | 保留筛选条件，提示 `复核单列表加载失败，请重试` |
| 无复核权限 | 隐藏开始/继续复核，仅允许查看详情 |
