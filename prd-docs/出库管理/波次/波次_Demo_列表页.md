# 波次_Demo_列表页

> 角色：Demo·列表页 | 类型：业务单据
> 页面只描述结构、控件、交互和 Mock；字段定义以 `波次字段清单.md` 为准，不重复维护字段枚举。

## 1. 页面定位

波次列表页用于仓库主管查看系统波次和手动波次，筛选待处理波次，并进入新增编辑页、详情页或执行分配/下推/关闭等组织动作。

## 2. 页面入口

| 入口 | 说明 |
|:--|:--|
| 菜单 | 出库管理 / 波次 |
| 关联跳转 | 出库管理概览、拣货单列表、异常订单池 |

## 3. 页面结构

```text
波次列表
├─ 顶部筛选区
│  ├─ 关键词：波次单号/承运商/线路/拣货员
│  ├─ 波次类型、状态、仓库、承运商、线路、优先级
│  ├─ 创建时间范围
│  └─ 查询、重置
├─ 操作区
│  ├─ 新建手动波次
│  └─ 触发系统波次（Demo 可见，生产是否开放需配置）
├─ 表格区
│  ├─ 波次单号、类型、状态、仓库、承运商、线路、优先级
│  ├─ 订单数、SKU数、总件数、拣货员、创建时间
│  └─ 行操作：分配拣货员、下推拣货、关闭、查看详情
└─ 分页
   └─ 默认 20 条/页
```

## 4. 行操作规则

| 场景 | 行操作 |
|:--|:--|
| 状态=草稿 | 分配人员、下推拣货、作废、查看详情 |
| 状态=拣货中 / 已拣货 / 已复核 | 查看详情、作废 |
| 状态=已交运 | 查看详情 |
| 状态=已作废 | 查看详情 |
| 按钮不可用 | 直接隐藏，不展示 disabled 态 |

## 5. 查询规则

- 关键词支持波次单号、承运商名称、线路名称、拣货员名称模糊查询。
- 状态筛选只使用波次自身状态。
- 创建时间按 `createdAt` 查询。
- 列表不展开商品明细；商品明细在详情页查看。

| 状态 | 展示 |
|:--|:--|
