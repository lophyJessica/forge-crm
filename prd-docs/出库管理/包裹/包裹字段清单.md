# 包裹字段清单

> 角色：字段清单(SSOT) | 类型：执行作业单
> 本文件为包裹字段唯一事实来源；主 PRD、业务规则、流程推演和 Demo 页面引用本文件，不重复维护字段口径。

## 1. 字段设计原则

- PKG 单号、来源 CHECK、来源 PICK、来源 WAVE、商品快照、出库数量不可手工编辑。
- 重量、承运商、面单号、面单打印状态是确认包装完成前的关键校验字段。
- 包装完成后，PKG 明细、重量、面单、承运商、库存过账结果均只读。
- 日期格式 `YYYY-MM-DD`，时间格式 `YYYY-MM-DD HH:mm:ss`。
- 空值展示 `-`，备注/异常说明 ≤200 字符。
- 库存三口径字段只记录包装完成前后快照，不允许人工编辑。

## 5. 枚举值

### 5.1 包裹状态 `status`

| 值 | 展示 | 说明 |
|:--|:--|:--|
| pending_pack | 待包装 | CHECK 已复核，等待包装 |
| packing | 包装中 | 已开始称重、面单或贴单作业 |
| packed | 已包装 | 包装完成，库存已过账，FL 已生成 |
| voided | 已作废 | 上游异常关闭且未包装完成 |

### 5.2 库存过账状态 `post_status`

| 值 | 展示 | 说明 |
|:--|:--|:--|
| not_posted | 未过账 | 尚未确认包装完成 |
| posted | 已过账 | 现存和占用已更新，FL 已生成 |
| post_failed | 过账失败 | 库存校验或 FL 生成失败，PKG 不得完成 |

### 5.3 面单打印状态 `waybill_print_status`

| 值 | 展示 | 说明 |
|:--|:--|:--|
| not_printed | 未打印 | 尚未打印面单 |
| printed | 已打印 | 已完成面单打印 |
| failed | 打印失败 | 打印失败，需重打 |

### 5.4 服务类型 `service_type`

| 值 | 展示 | 说明 |
|:--|:--|:--|
| standard | 标准 | 普通时效 |
| express | 加急 | 优先交运 |
| economy | 经济 | 低成本配送 |

## 6. 计算字段

| 字段 | 公式 | 说明 |
|:--|:--|:--|
| 出库总数 | `Σ outbound_qty` | 包裹内商品出库数量合计 |
| 已过账数量 | `Σ posted_qty` | 包装完成后应等于出库总数 |
| 过账后现存 | `过账前现存 - 出库数量` | 明细级库存实扣 |
| 过账后占用 | `过账前占用 - 出库数量` | 明细级释放占用 |
| 过账后可用 | `过账前可用` | 包装完成不再改变可用 |
| 完成条件 | `重量>0 且承运商存在 且面单号存在 且面单已打印且已贴单` | 满足后才进入库存过账 |

## 7. 校验时机

| 时机 | 校验范围 |
|:--|:--|
| 开始包装 | 来源 CHECK 是否已复核，PKG 是否待包装 |
| 重量录入 | 重量是否大于 0，格式是否为数字 |
| 面单生成/录入 | 面单号是否为空，是否与当前 PKG 绑定 |
| 打印面单 | 面单号存在且打印状态可更新 |
| 确认包装完成 | 重量、承运商、面单、贴单、明细、库存快照、幂等校验 |
| 库存过账 | 现存和占用是否足以覆盖出库数量，FL 单号是否可生成 |
