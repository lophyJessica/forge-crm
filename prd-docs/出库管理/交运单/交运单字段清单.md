# 交运单字段清单

> 角色：字段清单(SSOT) | 类型：执行作业单
> 本文件为交运单字段唯一事实来源；主 PRD、业务规则、流程推演和 Demo 页面引用本文件，不重复维护字段口径。

## 1. 字段设计原则

- DSH 单号、来源 PKG、包裹号、面单号、重量、承运商快照不可手工编辑。
- 承运商交接人、交接时间、实交包裹数、回传状态是确认交运和复核交接结果的关键字段。
- 交运确认后，交运头、明细和交接结果只读；回传失败只允许重试回传，不允许改交运结果。
- 日期格式 `YYYY-MM-DD`，时间格式 `YYYY-MM-DD HH:mm:ss`。
- 空值展示 `-`，备注/异常说明 ≤200 字符。
- DSH 不维护库存过账字段；如需查看库存实扣，应跳转上游 PKG/FL。

## 5. 枚举值

### 5.1 交运状态 `status`

| 值 | 展示 | 说明 |
|:--|:--|:--|
| PENDING_SHIP | 待交运 | 已由已包装包裹聚合，等待交接 |
| SHIPPING | 交运中 | 已开始扫描或填写交接信息 |
| SHIPPED | 已交运 | 已确认交接承运商，订单完结 |
| VOIDED | 已作废 | 未交运前任务异常关闭 |

### 5.2 扫描状态 `scanStatus`

| 值 | 展示 | 说明 |
|:--|:--|:--|
| PENDING | 待扫描 | 包裹尚未扫描 |
| SCANNED | 已扫描 | 包裹已成功扫描 |
| EXCEPTION | 异常 | 包裹少交、拒收或错扫记录 |

### 5.3 交接结果 `handoverResult`

| 值 | 展示 | 说明 |
|:--|:--|:--|
| PENDING | 待交接 | 尚未确认给承运商 |
| HANDED_OVER | 已交接 | 已交给承运商 |
| REJECTED | 承运商拒收 | 承运商现场拒收 |
| MISSING | 少交 | 应交但现场未交 |

### 5.4 回传状态 `*_sync_status`

| 值 | 展示 | 说明 |
|:--|:--|:--|
| NOT_SYNCED | 未回传 | 尚未执行回传/生成 |
| SYNCED | 已回传 | 回传或生成成功 |
| FAILED | 回传失败 | 可重试，不改变交运结果 |

### 5.5 异常原因 `exceptionReason`

| 值 | 展示 | 说明 |
|:--|:--|:--|
| WRONG_CARRIER | 承运商不一致 | 包裹承运商与交运单不一致 |
| NOT_PACKED | 未包装完成 | 包裹状态不是已包装 |
| REJECTED_BY_CARRIER | 承运商拒收 | 承运商现场拒收 |
| MISSING_PACKAGE | 少交 | 应交包裹现场缺失 |
| BARCODE_UNREADABLE | 条码不可识别 | 包裹号/面单无法识别 |
| OTHER | 其他 | 需填写行备注 |

## 6. 计算字段

| 字段 | 公式 | 说明 |
|:--|:--|:--|
| 应交包裹数 | `COUNT(pkgNo)` | 交运任务包裹总数 |
| 实交包裹数 | `COUNT(handoverResult=HANDED_OVER)` | 已确认交接包裹数 |
| 异常包裹数 | `COUNT(scanStatus=EXCEPTION or handoverResult in (rejected, missing))` | 交运异常数量 |
| 总重量 | `Σ packageWeightKg` | 包裹重量合计 |
| 交运完成条件 | `实交包裹数=应交包裹数 且 异常包裹数=0 且交接信息完整` | 满足后才允许确认交运 |

## 7. 校验时机

| 时机 | 校验范围 |
|:--|:--|
| PKG 聚合生成 | 包裹是否已包装、承运商是否一致、是否未被其他有效 DSH 聚合 |
| 开始交运 | 当前用户是否有发货权限，DSH 是否待交运 |
| 扫包裹/面单 | 是否属于当前 DSH，包裹状态是否已包装，承运商是否一致 |
| 确认交运 | 包裹数量、交接人、交接时间、交接地点是否满足 |
| 回传重试 | DSH 是否已交运，回传状态是否失败 |
