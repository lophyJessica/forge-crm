# 公海字段清单

> **说明**：公海是线索的子视图，字段来自《线索字段清单》。本文件仅列出公海视图独有的展示规则。
> **版本**：V1.0 | 2026-07-18

---

## 一、公海视图列表列

继承线索字段清单中 `列表展示=是` 的字段，并额外展示：

| 字段名称 | 字段来源 | 列表展示 | 说明 |
|----------|---------|:------:|------|
| 线索编号 | 继承线索 | 是 | |
| 公司名称 | 继承线索 | 是 | |
| 线索来源 | 继承线索 | 是 | |
| 所属行业 | 继承线索 | 是 | |
| AI评分 | 继承线索 | 是 | |
| 入池时间 | 系统计算 | 是 | PENDING_ASSIGN=创建时间, ABANDONED=放弃时间 |
| 放弃原因 | 继承线索 | 是 | 仅 ABANDONED 入池的展示 |
| 操作 | — | 是 | 认领按钮 |

---

## 二、入池类型标签

| 入池类型 | 标签文案 | 颜色 |
|----------|---------|------|
| NEW | 新线索 | 蓝色(待分配的原始线索) |
| RELEASED | 已释放 | 灰色(被放弃满7天的) |

---

## 三、排序规则

- 默认排序:AI评分倒序(高分线索优先)
- 认领后从公海移除,不再展示
