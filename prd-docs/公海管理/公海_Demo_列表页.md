# 公海_Demo_列表页

> **版本**：V1.0 | 2026-07-18

---

## 1. 页面概述

| 项目 | 内容 |
|------|------|
| 页面名称 | 公海 |
| 页面目标 | 展示所有可认领的线索,销售主动认领 |
| 页面路径 | `/lead-pool` |
| 骨架结构 | 查询栏 + 表格 |

---

## 2. 查询栏

| 字段 | 控件 | 说明 |
|------|------|------|
| 关键词 | Input | 公司名/手机号搜索 |
| 线索来源 | Select | |
| 所属行业 | Select | |
| AI评分 | Range | 0-100 |
| 入池类型 | Select | NEW=新线索 / RELEASED=已释放 |

---

## 3. 表格列

| 列名 | 宽度 | 说明 |
|------|------|------|
| 线索编号 | 160px | 链接色→详情页 |
| 公司名称 | 180px | |
| 线索来源 | 100px | |
| 所属行业 | 120px | |
| AI评分 | 80px | Tag配色 |
| 入池类型 | 100px | NEW蓝色/RELEASED灰色 |
| 入池时间 | 160px | |
| 操作 | 100px | 认领按钮 |

---

## 4. 认领交互

- 点击认领:无二次确认,状态→ASSIGNED,归属人=当前用户,从公海列表移除
- 认领成功:Toast"线索已认领,请及时跟进"

---

## 5. Mock 数据

≥ 10 条:5条 NEW(PENDING_ASSIGN),5条 RELEASED(ABANDONED满7天),AI评分 30-85 分布
