# 拜访计划_用例数据推演

> **版本**：V1.0 | 2026-07-18

## 案例1:正常流程

| 步骤 | 操作 | 数据变化 | 验证 |
|------|------|---------|------|
| 1 | 创建拜访,关联商机OPP001,上门拜访 | `状态=PLANNED` | |
| 2 | 到达现场,签到 | `状态:PLANNED→CHECKED_IN` `签到时间=当前时间` | |
| 3 | 填写拜访内容 | `状态:CHECKED_IN→COMPLETED` | 内容必填 |

## 案例2:取消

| 步骤 | 操作 | 预期 |
|------|------|------|
| 1 | PLANNED状态取消 | `状态→CANCELLED` |
| 2 | COMPLETED状态取消 | **阻断**:"已完成的拜访不可取消" |

| 步骤 | 操作 | 预期 |
