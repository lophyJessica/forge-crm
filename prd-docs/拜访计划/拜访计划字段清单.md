# 拜访计划字段清单

> **版本**：V1.0 | 2026-07-18

## 基础字段

| 字段名称 | 字段来源 | 取值说明 | 必填性 | 列表展示 |
|----------|---------|---------|:------:|:------:|
| 拜访编号 | 系统生成 | `VS{YYYYMMDD}-{4位序号}` | 系统自动 | 是 |
| 关联对象类型 | 用户选择/继承 | 枚举值: `LEAD` / `OPPORTUNITY` / `CUSTOMER` | 必填 | 是 |
| 关联对象ID | 系统带出/继承 | 对应关联实体的唯一编码 | 必填 | 是 |
| 拜访标题 | 用户填写 | 文本,上限100字符 | 必填 | 是 |
| 拜访方式 | 用户填写 | 上门/电话/视频 | 必填 | 是 |
| 计划拜访时间 | 用户填写 | `YYYY-MM-DD HH:mm` | 必填 | 是 |
| 拜访地址 | 用户填写 | 文本,选填 | 选填 | 否 |
| 拜访状态 | 系统生成 | PLANNED/CHECKED_IN/COMPLETED/CANCELLED | 系统自动 | 是 |
| 签到时间 | 系统生成 | `YYYY-MM-DD HH:mm` | 系统自动 | 是 |
| 签到地址 | 系统生成 | 签到时系统定位自动获取(如: 强盛科技科技园B座1楼) | 系统自动(签到后必填) | 否(详情展示是) |
| 拜访内容 | 用户填写 | Textarea, CHECKED_IN→COMPLETED 时必填 | 条件必填 | 否 |

## 状态Tag配色

| PLANNED=蓝色 | CHECKED_IN=橙色 | COMPLETED=绿色 | CANCELLED=红色 |
