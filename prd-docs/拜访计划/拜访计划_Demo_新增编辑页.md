# 拜访计划_Demo_新增编辑页

> **版本**：V2.0 | 2026-07-18
> **页面类型**：PC Web 新增页 / 编辑页
> **前置文档**：《拜访计划主PRD》《拜访计划字段清单》
> **字段约束**：字段定义、必填性、格式、长度和取值以《拜访计划字段清单》为唯一权威源。

---

## 1. 页面概述

| 项目 | 新增模式 | 编辑模式 |
|------|----------|----------|
| 页面名称 | 新建拜访 | 编辑拜访 |
| 页面路径 | `/visits/new` | `/visits/{id}/edit` |
| 进入方式 | 拜访列表、线索/商机/客户详情快捷入口 | 列表或详情“编辑” |
| 页面目标 | 创建一条已计划拜访 | 修改尚未签到的拜访计划 |
| 可用状态 | 新建 | 仅已计划 |
| 页面骨架 | 页头 + 来源提示 + 表单 + 固定底栏 | 页头 + 状态摘要 + 表单 + 固定底栏 |

页面原则：

- 状态不在表单直接修改。
- 一次拜访只关联一个对象。
- 关联类型和对象 ID 必须成对一致。
- 快捷创建时关联对象默认锁定。
- 保存创建计划，不执行签到。
- 编辑保存不改变状态。
- 签到信息由签到动作写入。
- 拜访内容由完成动作采集。
- 不提供删除。
- 表单最大宽度 960px。

## 2. 路由与初始化

### 2.1 从列表进入新增页

- 加载当前用户新增权限。
- 关联对象类型为空。
- 关联对象 ID 为空并禁用。
- 其他业务字段为空。
- 不预生成拜访编号。
- 不展示拜访状态。
- 不展示签到信息。
- 不展示拜访内容。

### 2.2 从关联对象详情进入

- 路由携带对象类型和对象标识。
- 后台校验关联对象存在。
- 后台校验当前用户可见。
- 后台校验对象允许创建拜访。
- 自动预填关联对象类型。
- 自动预填关联对象 ID。
- 两个字段默认锁定。
- 展示来源提示条。
- 可通过“重新选择”解除预填。

### 2.3 编辑模式初始化

- 根据路径标识加载拜访计划。
- 校验查看权限。
- 校验编辑权限。
- 校验状态仍为已计划。
- 展示拜访编号与状态只读摘要。
- 加载表单原值。
- 已签到或终态直接跳转详情。
- Toast `当前拜访状态不可编辑`。

### 2.4 路由参数

| 参数 | 来源 | 用途 | 异常处理 |
|------|------|------|----------|
| `id` | 编辑路径 | 查询拜访计划 | 不存在进入 404 |
| `objectType` | 快捷入口 | 预填关联类型 | 非法值忽略并提示 |
| `objectId` | 快捷入口 | 预填关联对象 | 无权或失效时阻断 |
| `from` | 来源页 | 返回上下文 | 非法值回拜访列表 |

| 字段名 | 关联对象ID |

## 5. 字段联动规则

| 编号 | 触发 | 目标 | 行为 |
|------|------|------|------|
| L01 | 选择关联对象类型 | 关联对象ID | 清空旧值并切换数据源 |
| L02 | 选择关联对象ID | 拜访标题 | 空值时生成建议 |
| L03 | 从详情快捷进入 | 类型与ID | 自动预填并锁定 |
| L04 | 重新选择来源对象 | 类型与ID | 二次确认后清空重选 |
| L05 | 拜访方式变化 | 拜访地址 | 仅更新 Placeholder，不覆盖输入 |
| L06 | 计划时间变化 | 签到窗口 | 保存后由后台重算可签到时间 |
| L07 | 保存成功 | 关联对象详情 | 拜访区块新增一条计划 |
| L08 | 保存失败 | 关联对象 | 不写入任何关联或状态变化 |

联动约束：

- 不静默覆盖手工输入。
- 不复制对象主数据。
- 不通过表单推进关联对象状态。
- 外部查询失败保留已选对象摘要。
- 快捷预填仍需保存时实时校验。

## 9. 保存交互规范

### 9.1 校验顺序

1. 用户操作权限。
2. 编辑模式当前状态。
3. 字段清单校验。
4. 关联类型与对象一致性。
5. 关联对象存在与可见性。
6. 关联对象业务可用性。
7. 计划时间业务边界。
8. 版本并发校验。
9. 保存与建立关联。

### 9.2 校验失败

- 所有错误字段同时标红。
- 滚动并聚焦第一个错误字段。
- Toast `请检查并完善拜访计划`。
- 类型与对象不一致时清空对象并要求重选。
- 对象已失效时显示返回对应列表入口。
- 时间错误时定位到日期控件。

### 9.3 保存成功

新增：

- 生成拜访编号。
- 状态进入已计划。
- 在关联对象详情展示计划。
- Toast `拜访计划创建成功`。
- 跳转拜访详情。

编辑：

- 更新计划信息。
- 状态保持已计划。
- Toast `拜访计划已更新`。
- 返回拜访详情。

### 9.4 保存失败

- 保留所有输入。
- 不创建孤立拜访。
- 不改变关联对象数据。
- 网络结果不确定时按请求键查询。
- 并发冲突 Toast `拜访计划已被他人更新`。
- 编辑期间被签到时跳转详情查看最新状态。

- 错误文案通过 aria-describedby 关联。
- SegmentedControl 支持方向键。
- Select 支持键盘搜索。
- 日期选择器支持键盘输入。
- 保存状态通过 aria-live 播报。
- 固定底栏不遮挡表单末尾。
