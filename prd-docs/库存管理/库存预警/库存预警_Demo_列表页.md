# 库存预警_Demo_列表页

> 状态：V1.0 已补充 | 角色：Demo·列表页 | 类型：查询页 | Mock 日期：2026-07

## 1. 页面定位

库存预警 Demo 仅实现列表查询页，用于展示低于安全库存/最低库存的库存记录。页面不提供新增、编辑、详情、确认、补货入口。

建议页面名：库存预警

建议路由：`/inventory/warnings`

终端：PC/PDA 共用口径，PC 优先表格，PDA 可使用紧凑卡片。

## 2. 页面结构

```text
库存预警
低于安全库存标黄，低于最低库存标红

[仓库 Select] [商品 Input] [预警等级 Select] [重置] [查询]

表格：
预警等级 | 仓库 | 商品编码 | 商品名称/规格 | 当前现存 | 安全库存 | 最低库存 | 单位 | 库存快照时间
```

## 3. 查询区

| 控件 | 字段 | 默认值 | 交互 |
|:--|:--|:--|:--|
| 仓库 Select | `warehouseCode` | 全部仓库 | 选择仓库后点击查询生效 |
| 商品 Input | `productKeyword` | 空 | 支持商品编码/名称关键词 |
| 预警等级 Select | `warningLevel` | `ALL` | 选项：全部预警、低于安全库存、低于最低库存 |
| 重置 Button | - | - | 清空筛选条件，不改变数据 |
| 查询 Button | - | - | 按筛选条件刷新列表 |

## 4. 列表区

| 列名 | 字段 | 展示规则 |
|:--|:--|:--|
| 预警等级 | `warningLevel` | `MINIMUM_LOW` 红色标签；`SAFETY_LOW` 黄色标签 |
| 仓库 | `warehouseName` | 展示仓库名称，悬浮/小字可展示 `warehouseCode` |
| 商品编码 | `productCode` | 等宽字体 |
| 商品名称/规格 | `productName`、`productSpec` | 名称主展示，规格小字展示；空值为 `-` |
| 当前现存 | `currentStock` | 数量右对齐 |
| 安全库存 | `safetyStock` | 数量右对齐；未配置为 `-` |
| 最低库存 | `minimumStock` | 数量右对齐；当前为商品档案占位字段，未配置为 `-` |
| 单位 | `unit` | 普通文本 |
| 库存快照时间 | `stockSnapshotAt` | `YYYY-MM-DD HH:mm:ss` |

## 5. 视觉规则

- `MINIMUM_LOW`：红色标签，行内关键数量可使用红色强调。
- `SAFETY_LOW`：黄色标签，行内关键数量可使用黄色强调。
- 红色优先级高于黄色，同一行只展示一个标签。
- 空值统一展示 `-`。
- 不展示正常库存行。
- 不展示状态 Tab。

## 7. 空状态与异常

| 场景 | 文案/表现 |
|:--|:--|
| 无匹配预警 | 暂无符合条件的库存预警 |
| 阈值缺失 | 阈值字段展示 `-`，不自动补值 |
| 查询失败 | 查询失败，请稍后重试 |
| PDA 离线 | 不新增自动推送；如展示缓存数据，需展示库存快照时间 |

## 8. 页面禁止项

- 禁止新增/编辑/详情页入口。
- 禁止“确认预警”“处理完成”“忽略预警”等动作。
- 禁止补货建议、补货入口、自动生成采购/调拨。
- 禁止在本页维护安全库存/最低库存。
