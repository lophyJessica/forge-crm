# 库存预警字段清单

> 状态：V1.0 已补充 | 角色：字段清单(SSOT) | 类型：查询页 | Mock 日期：2026-07
> 说明：本页只定义库存预警查询页字段，不定义商品档案维护字段。

## 1. SSOT 说明

| 数据 | 来源 | 说明 |
|:--|:--|:--|
| 当前现存 | 库存聚合结果 | 按仓库 + 商品汇总 `qtyTotal`，不包含在途 |
| 安全库存 | 商品档案 | `safetyStock` 为商品档案字段，上游 SSOT |
| 最低库存 | 商品档案 | `minimumStock` 为商品档案字段；当前商品档案字段清单仍为占位，字段编码待确认 |
| 预警等级 | 页面计算 | 不落库，不回写业务单据 |

## 2. 查询条件

| 字段名 | 中文名 | 类型/组件 | 必填 | 默认值 | 数据来源/枚举 | 校验与交互 |
|:--|:--|:--|:--|:--|:--|:--|
| `warehouseCode` | 仓库 | Select | 否 | 全部仓库 | 仓库档案启用数据 | 选择后按仓库过滤；空值表示全部 |
| `productKeyword` | 商品 | Input | 否 | 空 | 商品编码/商品名称关键词 | 支持商品编码、商品名称模糊匹配；前后空格自动忽略 |
| `warningLevel` | 预警等级 | Select | 否 | `ALL` | `ALL`、`SAFETY_LOW`、`MINIMUM_LOW` | `ALL` 表示展示全部预警等级；不包含正常库存 |

## 3. 列表展示

| 字段名 | 中文名 | 类型 | 必填 | 展示规则 | 来源/计算 |
|:--|:--|:--|:--|:--|:--|
| `warningLevel` | 预警等级 | Enum | 是 | `SAFETY_LOW` 标黄；`MINIMUM_LOW` 标红 | 按当前现存与商品档案阈值计算 |
| `warehouseCode` | 仓库编码 | String | 是 | 等宽编码展示 | 库存聚合维度 |
| `warehouseName` | 仓库名称 | String | 是 | 普通文本 | 仓库档案快照/库存聚合结果 |
| `productCode` | 商品编码 | String | 是 | 等宽编码展示 | 商品档案 |
| `productName` | 商品名称 | String | 是 | 普通文本 | 商品档案 |
| `productSpec` | 规格型号 | String | 否 | 空值展示 `-` | 商品档案 |
| `unit` | 计量单位 | String | 是 | 普通文本 | 商品档案 |
| `currentStock` | 当前现存 | Integer | 是 | 右对齐；低于最低库存时随行标红 | 按仓库 + 商品汇总 `qtyTotal` |
| `safetyStock` | 安全库存 | Integer | 否 | 右对齐；未配置展示 `-` | 商品档案 `safetyStock` |
| `minimumStock` | 最低库存 | Integer | 否 | 右对齐；未配置展示 `-` | 商品档案占位字段，字段编码待确认 |

## 4. 系统字段

| 字段名 | 中文名 | 类型 | 展示位置 | 来源/说明 |
|:--|:--|:--|:--|:--|
| `stockSnapshotAt` | 库存快照时间 | DateTime | 列表可选展示/调试说明 | 库存聚合结果最近更新时间；格式 `YYYY-MM-DD HH:mm:ss` |
| `thresholdSource` | 阈值来源 | String | 列表可选展示/调试说明 | 固定展示“商品档案”；最低库存未落地时标注“商品档案占位” |
| `computedAt` | 预警计算时间 | DateTime | 页面说明/调试说明 | 本次查询计算时间，不落库 |

## 5. 枚举定义

| 枚举 | 中文含义 | 展示 |
|:--|:--|:--|
| `ALL` | 全部预警 | 查询条件使用 |
| `SAFETY_LOW` | 低于安全库存 | 黄色标签 |
| `MINIMUM_LOW` | 低于最低库存 | 红色标签 |

## 6. 字段约束

- 数量字段为整数，展示口径遵循通用字段校验规范中的数量规则。
- `currentStock` 为库存结果字段，本页不可编辑。
- `safetyStock`、`minimumStock` 为商品档案字段，本页不可编辑。
- 阈值缺失时，本页不补录、不推导；对应字段展示 `-`，并按已配置阈值计算可计算的预警等级。
- 商品档案维护端应保证 `minimumStock <= safetyStock`；库存预警页只读展示，不负责维护校验。
