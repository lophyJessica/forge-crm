# 库存预警_用例数据推演

> 状态：V1.0 已补充 | 角色：用例数据推演 | 类型：查询页 | Mock 日期：2026-07

## 1. 数据来源说明

以下用例优先复用前端 Demo 已有的 2026 年仓库、商品、库存种子数据：

- 仓库：`WH001` 北京主仓、`WH002` 上海分仓、`WH003` 广州越秀仓、`WH004` 成都温江仓、`WH006` 武汉汉阳仓。
- 商品：`SKU001` 至 `SKU007`。
- 当前现存：来自库存 Mock 中的 `qtyTotal`。
- 安全库存：以商品档案 `safetyStock` 为 SSOT；若库存快照中存在冗余值，以商品档案为准。

不确定性：当前商品档案字段清单仍为占位，前端 Demo 未见 `minimumStock` 落地字段。下表中的 `minimumStock` 为商品档案占位字段样例，仅用于测试标红规则；待商品档案 PRD 确认后替换为正式字段与正式样例。

## 3. 用例 1：默认查询全部预警

| 项目 | 内容 |
|:--|:--|
| 前置条件 | 库存与商品档案样例数据已初始化 |
| 查询条件 | `warehouseCode` 空、`productKeyword` 空、`warningLevel=ALL` |
| 操作 | 用户点击查询 |
| 预期结果 | 返回 4 行：`WH003/SKU005`、`WH002/SKU001` 标红；`WH002/SKU003`、`WH004/SKU006` 标黄 |
| 不应出现 | 正常库存行、新增/编辑/详情/确认/补货按钮 |

## 4. 用例 2：筛选最低库存预警

| 项目 | 内容 |
|:--|:--|
| 查询条件 | `warningLevel=MINIMUM_LOW` |
| 操作 | 用户点击查询 |
| 预期结果 | 仅返回 `currentStock < minimumStock` 的红色预警行 |
| 样例结果 | `WH003/SKU005`、`WH002/SKU001` |
| 校验点 | 黄色预警 `WH002/SKU003`、`WH004/SKU006` 不展示 |

## 5. 用例 3：筛选安全库存预警

| 项目 | 内容 |
|:--|:--|
| 查询条件 | `warningLevel=SAFETY_LOW` |
| 操作 | 用户点击查询 |
| 预期结果 | 仅返回低于安全库存但未低于最低库存的黄色预警行 |
| 样例结果 | `WH002/SKU003`、`WH004/SKU006` |
| 校验点 | 红色预警优先级更高，不混入黄色结果 |

## 6. 用例 4：仓库 + 商品组合筛选

| 项目 | 内容 |
|:--|:--|
| 查询条件 | `warehouseCode=WH002`、`productKeyword=SKU003`、`warningLevel=ALL` |
| 操作 | 用户点击查询 |
| 预期结果 | 返回 1 行 `WH002/SKU003` |
| 展示 | `currentStock=15`、`safetyStock=20`、`minimumStock=10`、`warningLevel=SAFETY_LOW` |

## 7. 用例 5：边界值等于安全库存

| 项目 | 内容 |
|:--|:--|
| 样例数据 | `WH001/SKU002` 当前现存 30，安全库存 30，最低库存 10 |
| 查询条件 | `productKeyword=SKU002` |
| 预期结果 | 不进入预警清单 |
| 校验点 | 规则为严格小于 `<`，不是小于等于 `<=` |

## 8. 用例 6：边界值等于最低库存

| 项目 | 内容 |
|:--|:--|
| 样例数据 | `WH004/SKU006` 当前现存 5，最低库存 5，安全库存 15 |
| 查询条件 | `productKeyword=SKU006` |
| 预期结果 | 展示 `SAFETY_LOW`，不展示 `MINIMUM_LOW` |
| 校验点 | `currentStock = minimumStock` 不触发红色；但仍满足 `currentStock < safetyStock` |

## 9. 用例 7：最低库存字段未落地

| 项目 | 内容 |
|:--|:--|
| 前置条件 | 商品档案仅返回 `safetyStock`，未返回 `minimumStock` |
| 查询条件 | 任意 |
| 预期结果 | 不计算 `MINIMUM_LOW`；命中安全库存时展示 `SAFETY_LOW` |
| 页面展示 | `minimumStock` 展示 `-` 或“商品档案占位/待确认” |
| 禁止行为 | 不在库存预警页补录最低库存，不自动生成默认最低库存 |
