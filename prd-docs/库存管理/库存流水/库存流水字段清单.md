## 1. 文档定位

本文件是库存流水查询页字段的唯一定义来源。下游文档只引用字段名称、字段编码和枚举值，不重复定义字段含义。

本模块为只读查询页，因此字段清单只分为：

- 查询条件
- 列表展示
- 系统字段

不定义新增页、编辑页、详情页字段。

---

## 2. 术语口径

| 口径 | 字段编码 | 含义 | 公式关系 |
| :--- | :--- | :--- | :--- |
| 现存 | `qty_on_hand` | 仓内实物账面数量 | `现存 = 期初 + Σ入库 - Σ出库 + Σ盘盈 - Σ盘亏` |
| 占用 | `qty_allocated` | 已被销售订单等业务锁定的数量 | 参与可用计算 |
| 冻结 | `qty_frozen` | 质检、盘点、报损等场景锁定的数量 | 参与可用计算 |
| 可用 | `qty_available` | 可销售、可出库数量 | `可用 = 现存 - 占用 - 冻结` |

---

## 3. 枚举定义

### 3.1 变动类型 `change_type`

| 枚举值 | 展示文案 | 展示方向 | 影响口径 | 说明 |
| :--- | :--- | :---: | :--- | :--- |
| `INBOUND` | 入库+ | `+N` | 现存增加；可用增加或冻结转可用，按入库链确认口径写入 | 入库/上架确认类库存增加 |
| `OUTBOUND` | 出库- | `-N` | 现存减少，占用释放 | 出库实扣，按包装完成触发 |
| `TRANSFER_IN` | 调拨+ | `+N` | 调入仓现存增加，在途减少 | 调拨入库确认 |
| `TRANSFER_OUT` | 调拨- | `-N` | 调出仓现存减少，在途增加 | 调拨出库确认 |
| `STOCK_GAIN` | 盘盈+ | `+N` | 现存增加 | 盘点差异确认后调增 |
| `STOCK_LOSS` | 盘亏- | `-N` | 现存减少 | 盘点差异确认后调减 |
| `FREEZE` | 冻结 | `+N` | 冻结增加，可用减少，现存不变 | 质检、盘点、报损锁库 |
| `UNFREEZE` | 解冻 | `-N` | 冻结减少，可用增加，现存不变 | 质检合格、盘点完成等解除冻结 |
| `ALLOCATE` | 占用 | `+N` | 占用增加，可用减少，现存不变 | SO 审核等锁定库存 |
| `RELEASE` | 释放 | `-N` | 占用减少，可用增加，现存不变 | 订单取消或出库实扣后释放占用 |

### 3.2 变动方向 `change_direction`

| 枚举值 | 展示文案 | 匹配规则 |
| :--- | :--- | :--- |
| `ALL` | 全部 | 不过滤 |
| `INCREASE` | 增加 | `change_qty > 0` |
| `DECREASE` | 减少 | `change_qty < 0` |

> 说明：`FREEZE`、`ALLOCATE` 展示为 `+N`，表示冻结/占用口径增加；`UNFREEZE`、`RELEASE` 展示为 `-N`，表示冻结/占用口径减少。它们不代表现存增加或减少。

## 4. 查询条件

| 字段名称 | 字段编码 | 控件类型 | 取值来源/格式 | 必填 | 默认值 | 匹配规则 | 说明 |
| :--- | :--- | :--- | :--- | :---: | :--- | :--- | :--- |
| FL 单号 | `flow_no` | `Input` | `FL{YYYYMMDD}-{8位序号}` | 否 | 空 | 模糊匹配，系统转大写 | 用于直接定位某条流水 |
| 仓库 | `warehouse_code` | `SelectWithSearch` | 启用中的仓库档案；展示 `{warehouse_code} {warehouse_name}` | 否 | 全部可见仓库 | 精确匹配 | 受用户仓库权限范围限制 |
| 库区 | `zone_code` | `SelectWithSearch` | 启用中的库区；按所选仓库级联 | 否 | 全部库区 | 精确匹配 | 用于缩小到收货区、存储区、拣货区等 |
| 货位 | `location_code` | `Input` / `ScanInput` | 货位编码或 PDA 扫码结果 | 否 | 空 | 精确匹配优先，支持右模糊 | PDA 场景优先扫码 |
| 商品 | `product_keyword` | `SelectWithSearch` | 商品编码/名称/条码联想 | 否 | 全部商品 | 编码精确或名称模糊 | 商品信息来自商品档案快照 |
| 时间段 | `occurred_range` | `DateTimeRange` | `YYYY-MM-DD HH:mm:ss` | 否 | 最近 90 天 | 闭区间 | 查询跨度不得超过 365 天 |
| 变动类型 | `change_type` | `Select` | 见 `change_type` 枚举 | 否 | 全部类型 | 精确匹配 | 入库、出库、调拨、盘点、冻结等 |
| 变动方向 | `change_direction` | `Select` | `ALL` / `INCREASE` / `DECREASE` | 否 | `ALL` | 按 `change_qty` 正负过滤 | 辅助快速看增加或减少 |
| 来源单号 | `source_order_no` | `Input` | 来源业务单号，如 `PUT`、`PKG`、`TR`、`CK`、`BL`、上游 `SO` | 否 | 空 | 模糊匹配，系统转大写 | 用于从业务单据反查流水 |

## 5. 列表展示

| 字段名称 | 字段编码 | 字段来源 | 展示格式 | 建议宽度 | 是否默认展示 | 说明 |
| :--- | :--- | :--- | :--- | :---: | :---: | :--- |
| FL 单号 | `flow_no` | 系统生成 | `FL{YYYYMMDD}-{8位序号}`，如 `FL20260706-00000001` | 220px | 是 | 流水唯一标识，只读 |
| 发生时间 | `occurred_at` | 系统生成 | `YYYY-MM-DD HH:mm:ss` | 170px | 是 | 库存动作生效并写入流水的时间 |
| 仓库 | `warehouse_name` | 系统快照 | `{warehouse_code} {warehouse_name}` | 160px | 是 | 发生库存变动的仓库 |
| 库区 | `zone_name` | 系统快照 | `{zone_code} {zone_name}`；空值展示 `-` | 150px | 是 | 货位所属库区 |
| 货位 | `location_code` | 系统快照 | 货位编码，如 `LOC-A01` | 120px | 是 | 发生库存变动的具体货位 |
| 商品编码 | `product_code` | 商品快照 | 商品编码 | 120px | 是 | 发生变动的 SKU |
| 商品名称 | `product_name` | 商品快照 | 商品名称 | 180px | 是 | 商品名称快照 |
| 规格型号 | `product_spec` | 商品快照 | 空值展示 `-` | 140px | 是 | 商品规格快照 |
| 单位 | `unit` | 商品快照 | 商品计量单位 | 80px | 是 | 如 `盒`、`个`、`支` |
| 变动类型 | `change_type` | 系统生成 | 展示文案见枚举 | 120px | 是 | 说明本次变动业务类型 |
| 变动数量 | `change_qty` | 系统生成 | 带正负号整数，如 `+100`、`-20` | 110px | 是 | 展示本次变动数量；冻结/占用类表示对应口径的增减 |
| 变动后现存 | `qty_on_hand_after` | 系统快照 | 非负整数；倒挂场景允许小于 0 并红色展示 | 120px | 是 | 本次动作完成后的现存快照 |
| 来源单号 | `source_order_no` | 系统带出 | 来源业务单号；空值展示 `-` | 180px | 是 | 追溯触发本流水的业务单据或上游动作 |
| 操作人 | `operator_name` | 系统带出 | 用户姓名/系统任务名 | 120px | 是 | 触发库存动作的人员或系统任务 |

## 6. 系统字段

| 字段名称 | 字段编码 | 字段来源 | 必填 | 展示位置 | 说明 |
| :--- | :--- | :--- | :---: | :--- | :--- |
| 流水主键 | `flow_id` | 系统生成 | 是 | 不展示 | 数据库主键，可与 `flow_no` 相同 |
| 来源单据类型 | `source_order_type` | 系统带出 | 否 | 不展示 | 如 `PUT`、`PKG`、`TR`、`CK`、`BL`、`SO` |
| 来源单据行ID | `source_line_id` | 系统带出 | 否 | 不展示 | 用于保证同一来源行幂等生成 |
| 仓库编码 | `warehouse_code` | 系统快照 | 是 | 查询/列表组合展示 | 权限过滤与查询匹配使用 |
| 库区编码 | `zone_code` | 系统快照 | 否 | 查询/列表组合展示 | 货位所属库区 |
| 商品ID | `product_id` | 商品档案 | 是 | 不展示 | 商品主键 |
| 变动后占用 | `qty_allocated_after` | 系统快照 | 是 | 不默认展示 | 支持复核可用公式 |
| 变动后冻结 | `qty_frozen_after` | 系统快照 | 是 | 不默认展示 | 支持复核可用公式 |
| 变动后可用 | `qty_available_after` | 系统计算 | 是 | 不默认展示 | `qty_on_hand_after - qty_allocated_after - qty_frozen_after` |
| 创建时间 | `created_at` | 系统生成 | 是 | 不展示 | 流水落库时间，通常等于 `occurred_at` |
| 创建人 | `created_by` | 系统生成 | 是 | 不展示 | 触发库存动作的账号或系统任务 |

## 7. 校验与展示规则

| 规则 | 说明 |
| :--- | :--- |
| 单号规则 | `flow_no` 系统自动生成，不可编辑；格式为 `FL{YYYYMMDD}-{8位序号}` |
| 日期规则 | 查询时间跨度不得超过 365 天；日期时间格式 `YYYY-MM-DD HH:mm:ss` |
| 数量规则 | `change_qty` 为整数，展示时必须带正负号；数量来源由系统库存动作写入 |
| 空值规则 | 空值统一展示 `-` |
| 排序规则 | 默认按 `occurred_at` 倒序展示 |
| 分页规则 | PC 表格默认 20 条/页 |
| 只读规则 | 字段仅可查询和展示，不提供前端编辑控件 |
