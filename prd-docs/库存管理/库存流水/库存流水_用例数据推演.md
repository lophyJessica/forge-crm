## 1. 推演目标

验证库存流水查询页在以下方面符合规则：

- FL 单号格式正确
- 变动类型和正负号正确
- 按仓库、货位、商品、时间段、变动类型组合查询正确
- 变动后现存与三口径公式可复核
- 查询页保持只读，不引入新增、编辑、详情、调整动作

## 10. 推演结论

| 项目 | 结论 |
| :--- | :--- |
| FL 编码 | Mock 已使用 `FL{YYYYMMDD}-{8位序号}` |
| 变动类型 | 覆盖入库、出库、调拨、盘盈、盘亏、冻结、解冻、占用、释放 |
| 三口径 | 可通过系统字段快照按公式复核 |
| 查询页边界 | 仅列表查询，不引入新增、编辑、详情、调整 |
