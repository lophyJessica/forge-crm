## 1. 页面概述

| 项目 | 内容 |
| :--- | :--- |
| 页面名称 | 库存流水 |
| 页面类型 | 查询页 |
| 页面路径 | `/inventory/flows` |
| 终端 | PC / PDA |
| 页面目标 | 按仓库、货位、商品、时间段、变动类型查询只读库存变动流水 |
| 页面结构 | 查询区 + 工具条 + 列表区 |

本 Demo 只描述列表页。不设计新增页、编辑页、详情页，不提供库存调整入口。

---

## 2. 状态 Tab

**不适用**。

库存流水是只读审计日志，无草稿、待处理、已完成、作废等状态流转。页面不展示状态 Tab，不提供状态筛选。

---

## 3. 查询区

### 3.1 PC 布局

- 查询区采用 4 列栅格
- 常用条件默认展示：仓库、货位、商品、时间段、变动类型
- 次要条件可折叠展示：FL 单号、库区、来源单号、变动方向
- 操作按钮固定在查询区右下：`重置`、`查询`

### 3.2 PDA 布局

- 顶部优先展示 `ScanInput`：扫描货位/商品/来源单号
- 其次展示时间段和变动类型
- 查询结果以紧凑列表展示，字段优先级：发生时间、货位、商品、变动类型、变动数量、变动后现存

### 3.3 查询条件

| 字段 | 控件 | 展示要求 |
| :--- | :--- | :--- |
| FL 单号 | `Input` | 占位文案：`输入FL单号` |
| 仓库 | `SelectWithSearch` | 展示 `{warehouseCode} {warehouseName}`，按权限过滤 |
| 库区 | `SelectWithSearch` | 选择仓库后级联；未选仓库时展示全部可见库区 |
| 货位 | `Input` / `ScanInput` | PDA 支持扫码录入 |
| 商品 | `SelectWithSearch` | 支持商品编码、名称、条码检索 |
| 时间段 | `DateTimeRange` | 默认最近 90 天；跨度超过 365 天时提示 |
| 变动类型 | `Select` | 选项来自字段清单 `changeType` |
| 变动方向 | `Select` | `ALL` / `INCREASE` / `DECREASE` |
| 来源单号 | `Input` | 支持模糊查询，不触发来源单据动作 |

---

## 4. 工具条

| 区域 | 内容 | 规则 |
| :--- | :--- | :--- |
| 左侧 | 查询结果摘要 | 展示 `共 N 条流水`、当前筛选时间段 |
| 右侧 | `导出` | 有查询结果时展示；无结果时隐藏 |

禁止出现：

- 新增
- 编辑
- 删除
- 作废
- 库存调整
- 状态变更

---

## 5. 列表区

### 5.1 表格行为

| 行为 | 规格 |
| :--- | :--- |
| 默认排序 | `occurredAt DESC` |
| 分页 | PC 默认 20 条/页 |
| 行选择 | 不支持批量勾选 |
| 行操作 | 不提供行操作按钮 |
| 空状态 | 展示「暂无库存流水记录」 |
| 来源单号 | 只读文本，可复制；本 Demo 不提供来源详情跳转 |

### 5.2 列定义

| 列名 | 字段编码 | 宽度 | 固定 | 展示说明 |
| :--- | :--- | :---: | :---: | :--- |
| FL 单号 | `flNo` | 220px | 左固定 | `FL{YYYYMMDD}-{8位序号}` |
| 发生时间 | `occurredAt` | 170px | - | `YYYY-MM-DD HH:mm:ss` |
| 仓库 | `warehouseName` | 160px | - | `{warehouseCode} {warehouseName}` |
| 库区 | `zoneName` | 150px | - | 空值展示 `-` |
| 货位 | `locationCode` | 120px | - | 货位编码 |
| 商品编码 | `productCode` | 120px | - | SKU 编码 |
| 商品名称 | `productName` | 180px | - | 商品名称快照 |
| 规格型号 | `productSpec` | 140px | - | 空值展示 `-` |
| 单位 | `unit` | 80px | - | 计量单位 |
| 变动类型 | `changeType` | 120px | - | 入库+、出库-、调拨+、调拨- 等 |
| 变动数量 | `changeQty` | 110px | - | 正数带 `+`，负数带 `-` |
| 变动后现存 | `qtyOnHandAfter` | 120px | - | 本次变动后的现存快照 |
| 来源单号 | `sourceOrderNo` | 180px | - | 只读，可复制 |
| 操作人 | `operatorName` | 120px | - | 用户姓名或系统任务名 |

## 7. 反馈文案

| 场景 | 反馈方式 | 文案 |
| :--- | :--- | :--- |
| 查询成功 | 列表摘要 | `共查询到 N 条库存流水` |
| 查询无数据 | 空状态 | `暂无库存流水记录` |
| 时间跨度超限 | 字段提示 | `查询时间跨度不可超过 365 天` |
| 开始时间晚于结束时间 | 字段提示 | `开始时间不能晚于结束时间` |
| 查询失败 | Toast | `库存流水查询失败，请稍后重试` |
| 导出成功 | Toast | `库存流水导出成功` |
| 导出失败 | Toast | `库存流水导出失败，请稍后重试` |

## 8. 前端验收清单

| 检查项 | 预期 |
| :--- | :--- |
| 页面是否只有列表页 | 是 |
| 是否无状态 Tab | 是 |
| 是否无新增/编辑/详情/调整入口 | 是 |
| FL 单号是否为新格式 | 是，使用 `FL{YYYYMMDD}-{8位序号}` |
| 变动数量是否带正负号 | 是 |
| 冻结/占用类是否不误改现存展示 | 是，`qtyOnHandAfter` 按快照展示 |
| 空状态是否无新增引导 | 是 |
| 无数据时导出是否隐藏 | 是 |
