# 库存查询字段清单

> 状态：已补全
> 角色：字段清单(SSOT) | 类型：查询页
> 读者：研发工程师、测试工程师、产品复核
> 引用规则：本文件是库存查询字段的唯一定义来源；Demo 与用例只引用字段名，不重复定义字段本体。

## 1. 查询条件

| 字段名 | 字段编码 | 控件类型 | 数据类型 | 必填 | 默认值 | 取值/校验 | 筛选规则 | 说明 |
|:--|:--|:--|:--|:--:|:--|:--|:--|:--|
| 仓库 | warehouse_code | SelectWithSearch | string | 否 | 用户授权仓库范围 | 来源于启用仓库档案 | 精确匹配 | 仓管员仅展示授权仓库；权限模型未明确时以系统权限配置为准 |
| 库区 | zone_code | SelectWithSearch | string | 否 | 全部 | 来源于所选仓库下启用库区 | 精确匹配 | 未选择仓库时展示用户授权仓库范围内库区 |
| 货位 | location_code | ScanInput/Input | string | 否 | 空 | 支持货位编码或货位条码；PDA 扫码优先 | 精确匹配 | 必须属于所选仓库/库区；无权货位不返回 |
| 商品 | sku_keyword | SelectWithSearch/Input | string | 否 | 空 | 支持商品编码、商品名称、商品条码 | 编码/条码精确，名称模糊 | PDA 可扫码商品条码 |
| 批次 | batch_no | Input | string | 否 | 空 | 文本，建议不超过 50 字符 | 模糊匹配 | 未启用批次商品列表展示 `-` |
| 不显示零库存 | hide_zero_stock | Switch | boolean | 否 | false | `true`/`false` | 数量过滤 | `true` 时隐藏 `current_qty <= 0` 的行 |
| 每页条数 | page_size | Select | integer | 否 | 20 | PC 支持 20/50/100；PDA 可用 20 | 分页参数 | 遵守 PC 表格默认 20 条 |

## 2. 列表展示

默认列表粒度为 `warehouse_code + zone_code + location_code + sku_code + batch_no`。数量字段均为整数，右对齐展示。

| 字段名 | 字段编码 | 数据类型 | 字段来源 | 展示 | 宽度建议 | 规则/公式 | 说明 |
|:--|:--|:--|:--|:--:|:--|:--|:--|
| 仓库编码 | warehouse_code | string | 仓库档案 | 是 | 120px | - | 仓库唯一编码 |
| 仓库名称 | warehouse_name | string | 仓库档案 | 是 | 140px | - | 展示仓库中文名称 |
| 库区编码 | zone_code | string | 库区档案 | 是 | 120px | - | 库区唯一编码 |
| 库区名称 | zone_name | string | 库区档案 | 是 | 120px | - | 如存储区、拣货区、收货区 |
| 货位编码 | location_code | string | 货位档案/库存余额 | 是 | 120px | - | 如 `A-01-02` |
| 货位条码 | location_barcode | string | 货位档案 | 否 | 140px | 空值展示 `-` | PDA 扫码校验使用 |
| 商品编码 | sku_code | string | 商品档案快照 | 是 | 120px | - | 商品唯一编码 |
| 商品名称 | sku_name | string | 商品档案快照 | 是 | 180px | - | 商品中文名称 |
| 商品条码 | sku_barcode | string | 商品档案快照 | 否 | 140px | 空值展示 `-` | PDA 扫码校验使用 |
| 规格型号 | spec | string | 商品档案快照 | 是 | 140px | 空值展示 `-` | 商品规格 |
| 单位 | unit | string | 商品档案快照 | 是 | 80px | 空值展示 `-` | 件、箱、卷等 |
| 批次 | batch_no | string | 库存余额/流水 | 是 | 140px | 空值展示 `-` | 启用批次时按批次拆行 |
| 现存 | current_qty | integer | 系统计算 | 是 | 100px | `期初 + Σ入库 - Σ出库 + Σ盘盈 - Σ盘亏` | 当前货位实物数量口径 |
| 占用 | occupied_qty | integer | 系统计算 | 是 | 100px | 来源于销售订单审核等锁库结果 | 被销售或出库需求锁定的数量 |
| 可用 | available_qty | integer | 系统计算 | 是 | 100px | `current_qty - occupied_qty - frozen_qty` | 可销售/可出库数量 |
| 冻结 | frozen_qty | integer | 系统计算 | 是 | 100px | 质检、盘点、报损等触发 | 冻结数量不可出库 |
| 在途 | in_transit_qty | integer | 系统计算 | 是 | 100px | 调拨途中数量 | 不参与可用公式 |
| 最近流水号 | last_flow_no | string | 库存流水 FL | 是 | 170px | 格式 `FL{YYYYMMDD}-{8位序号}` | 无流水展示 `-` |
| 最近变动时间 | last_changed_at | datetime | 库存流水 FL | 是 | 170px | `YYYY-MM-DD HH:mm:ss` | 按通用时间格式展示 |

## 3. 系统字段

| 字段名 | 字段编码 | 数据类型 | 字段来源 | 必填 | 展示位置 | 规则/说明 |
|:--|:--|:--|:--|:--:|:--|:--|
| 记录唯一键 | inventory_key | string | 系统生成 | 是 | 隐藏 | 建议由 `warehouse_code\|zone_code\|location_code\|sku_code\|batch_no` 组成 |
| 查询时间 | queried_at | datetime | 系统生成 | 是 | 列表底部/接口返回 | `YYYY-MM-DD HH:mm:ss`，表示本次查询快照时间 |
| 当前页 | page_no | integer | 查询参数 | 是 | 分页器 | 从 1 开始 |
| 每页条数 | page_size | integer | 查询参数 | 是 | 分页器 | 默认 20 |
| 总行数 | total_count | integer | 查询结果 | 是 | 分页器 | 当前筛选条件下的总记录数 |
| 权限范围 | permission_scope | object | 权限系统 | 是 | 隐藏 | 用于后端过滤授权仓库/库区/货位 |
| 数据刷新标识 | data_version | string | 系统生成 | 否 | 隐藏 | 用于研发排查缓存或重复查询问题 |

## 4. 展示约束

- 本字段清单不设置新增页、编辑页、详情页列。
- 数量字段必须同屏展示 `现存`、`占用`、`可用`，并补充展示 `冻结`、`在途`。
- `available_qty` 必须按 `current_qty - occupied_qty - frozen_qty` 校验；如接口返回与公式不一致，前端不得自行改库存，应提示数据异常并记录日志。
- 空值统一展示 `-`。
- 本页面无状态字段、无状态 Tab、无状态机流转字段。
