# 合同_Demo_新增编辑页

> **版本**：V2.0 | 2026-07-18
> **页面类型**：PC Web 新增页 / 编辑页
> **前置文档**：《合同主PRD》《合同字段清单》
> **字段约束**：字段定义、必填性、格式和长度限制以《合同字段清单》为唯一权威源。

---

## 1. 页面概述

| 项目 | 新增模式 | 编辑模式 |
|------|----------|----------|
| 页面名称 | 新建合同 | 编辑合同 |
| 页面路径 | `/contracts/new` | `/contracts/{id}/edit` |
| 进入方式 | 合同列表、商机详情“发起合同” | 合同列表或详情“编辑” |
| 页面目标 | 创建合同草稿并建立有效商机绑定 | 修改草稿合同信息 |
| 可用状态 | 新建 | 仅草稿 |
| 页面骨架 | 页头 + 来源提示 + 表单 + 固定底栏 | 页头 + 状态提示 + 表单 + 固定底栏 |

页面原则：

- 合同状态不作为可编辑字段。
- 关联客户只从关联商机继承。
- 关联客户不可手工切换。
- 关联商机必须满足合同主 PRD 的签约条件。
- 同一商机只允许一个有效合同。
- 保存创建合同草稿，不等于提交签署。
- 编辑保存不改变合同状态。
- 合同不提供删除。
- 内容最大宽度 960px。

## 2. 路由与初始化

### 2.1 从合同列表进入新增页

- 页面加载新增权限。
- 关联商机为空。
- 关联客户为空且只读占位。
- 合同名称为空。
- 合同金额为空。
- 不预生成合同编号。
- 页面提示 `请选择可签约商机`。

### 2.2 从商机详情进入新增页

- 路由携带商机标识。
- 后台校验商机查看和合同创建权限。
- 校验商机处于允许发起合同的阶段。
- 校验商机没有其他有效合同。
- 自动预填关联商机。
- 自动继承关联客户。
- 自动预填可用的合同名称建议值。
- 自动预填商机金额作为合同金额建议值。
- 建议值仍需用户确认并保存。
- 关联商机默认锁定，可通过“重新选择”解除。

### 2.3 编辑页初始化

- 根据路径标识查询合同。
- 校验合同存在。
- 校验查看权限。
- 校验编辑权限。
- 校验合同仍为草稿。
- 展示合同编号和合同状态只读摘要。
- 加载表单原值。
- 不可编辑状态直接跳转详情页。
- Toast `当前合同状态不可编辑`。

### 2.4 路由参数

| 参数 | 来源 | 用途 | 异常处理 |
|------|------|------|----------|
| `id` | 编辑路径 | 查询合同 | 不存在进入 404 |
| `opportunityId` | 商机详情 | 预填关联商机 | 无权、无效或已有合同时阻断 |
| `from` | 来源页 | 返回恢复上下文 | 非法值回合同列表 |

## 4. 字段控件规格

> “限制与校验”只引用 SSOT，不在本文重复字段枚举、必填性或具体上限。

### 4.1 合同名称

| 项目 | 规格 |
|------|------|
| 字段名 | 合同名称 |
| 控件类型 | Input |
| Placeholder | `请输入合同名称` |
| 宽度 | 100% |
| 字符限制 | 引用《合同字段清单》 |
| 必填性 | 引用《合同字段清单》 |
| 清洗 | 失焦去除首尾空格 |
| 联动 | 不改变合同状态；从商机进入时可生成建议值 |

输入交互：

- 展示字符计数。
- 达到 SSOT 上限后禁止继续输入。
- 粘贴超限时截断并提示。
- 仅空格视为空值。
- 建议值可被用户修改。

### 4.2 关联商机

| 项目 | 规格 |
|------|------|
| 字段名 | 关联商机 |
| 控件类型 | SelectWithSearch |
| Placeholder | `请输入商机名称或编号搜索` |
| 宽度 | 100% |
| 数据来源 | CRM 商机查询 |
| 必填性 | 引用《合同字段清单》 |
| 搜索 | 300ms 防抖，远程查询 |
| 联动 | 选择后继承关联客户，并生成名称与金额建议 |

候选规则：

- 只返回操作者可见商机。
- 只返回满足合同主 PRD 签约条件的商机。
- 已绑定有效合同的商机不允许选择。
- 选项第一行展示商机名称。
- 选项第二行展示商机编号、客户和当前阶段。
- 无候选显示 `暂无可签约商机`。
- 不提供在下拉中新增商机。

选择行为：

- 选择后请求商机最新版本。
- 自动填充关联客户。
- 自动填充合同名称建议。
- 自动填充合同金额建议。
- 用户已手工修改名称或金额时不静默覆盖。
- 更换商机前弹二次确认。

### 4.3 关联客户

| 项目 | 规格 |
|------|------|
| 字段名 | 关联客户 |
| 控件类型 | ReadOnlySelect / Description |
| Placeholder | `选择商机后自动带出` |
| 宽度 | 100% |
| 数据来源 | 关联商机继承的 CRM 客户快照 |
| 必填性 | 引用《合同字段清单》 |
| 可编辑 | 否 |
| 联动 | 随关联商机更新；不能单独清空或更换 |

展示规则：

- 显示公司名称和客户编号。
- 点击链接进入客户详情。
- 客户快照失效时显示错误提示。
- 不提供新增客户。
- 不提供编辑客户。
- 保存前再次校验客户与商机一致。

### 4.4 合同金额

| 项目 | 规格 |
|------|------|
| 字段名 | 合同金额 |
| 控件类型 | NumberInput |
| Placeholder | `请输入合同金额` |
| 宽度 | 100% |
| 前缀 | `¥` |
| 数值与精度限制 | 引用《合同字段清单》 |
| 必填性 | 引用《合同字段清单》 |
| 联动 | 选择商机时带出预计金额建议；签署后作为成交与订单输入 |

金额交互：

- 禁止负数。
- 禁止科学计数法字符。
- 聚焦时显示原始值。
- 失焦后格式化。
- 与商机预计金额不一致时显示弱提示。
- 弱提示不自动覆盖输入。
- 保存时记录用户确认金额。

## 5. 字段联动规则

| 编号 | 触发 | 目标 | 行为 |
|------|------|------|------|
| L01 | 从商机详情进入 | 关联商机 | 自动预填并锁定 |
| L02 | 选择关联商机 | 关联客户 | 自动继承且只读 |
| L03 | 选择关联商机 | 合同名称 | 空值时生成建议，不覆盖手工值 |
| L04 | 选择关联商机 | 合同金额 | 空值时带出预计金额建议 |
| L05 | 更换关联商机 | 关联客户 | 先确认，再替换客户 |
| L06 | 更换关联商机 | 名称与金额 | 用户已修改时不自动覆盖 |
| L07 | 保存成功 | 商机阶段 | 建立绑定并联动进入合同阶段 |
| L08 | 保存失败 | 全部表单 | 保留输入且不改变商机 |

联动约束：

- 自动填充必须可解释。
- 不静默覆盖用户手工输入。
- 关联客户永远与商机客户一致。
- 商机阶段联动只在后端事务成功后生效。
- 前端预览不提前改变商机。

## 6. 更换商机确认

| 项目 | 规格 |
|------|------|
| 触发 | 已选择商机后选择另一商机 |
| 标题 | `更换商机` |
| 内容 | `更换后将同步更新关联客户，现有建议信息可能不再适用。` |
| 取消 | `继续编辑` |
| 确认 | `确认更换` |

确认后行为：

- 替换关联商机。
- 重新继承关联客户。
- 清空旧商机校验结果。
- 对用户已手工修改的合同名称不覆盖。
- 对用户已手工修改的合同金额不覆盖。
- 显示“请核对合同信息”弱提示。

## 7. 新增与编辑模式差异

| 能力 | 新增 | 草稿编辑 | 待签署及以上直接访问 |
|------|:----:|:--------:|:--------------------:|
| 合同名称 | 可编辑 | 可编辑 | 不允许进入 |
| 关联商机 | 可选择 | 默认锁定，不允许更换有效绑定 | 不允许进入 |
| 关联客户 | 自动继承只读 | 只读 | 不允许进入 |
| 合同金额 | 可编辑 | 可编辑 | 不允许进入 |
| 合同编号 | 不展示 | 只读 | 详情只读 |
| 合同状态 | 不展示 | 只读 | 详情只读 |
| 签署日期 | 不展示 | 只读 | 详情只读 |
| 保存按钮 | 创建合同 | 保存修改 | 不展示 |

编辑模式规则：

- 已建立有效绑定后不允许更换关联商机。
- 如需更换，必须作废当前合同并重新创建。
- 草稿保存不提交签署。
- 编辑保存后仍为草稿。
- 并发版本变化时禁止覆盖。

## 9. 保存交互规范

### 9.1 校验顺序

1. 当前用户权限。
2. 当前合同状态（编辑模式）。
3. 字段清单校验。
4. 商机最新状态校验。
5. 有效合同唯一性校验。
6. 关联客户一致性校验。
7. 合同版本并发校验。
8. 合同创建与商机联动事务。

### 9.2 校验失败

- 所有错误项同时标红。
- 页面滚动到第一个错误项。
- 第一个错误控件获得焦点。
- Toast `请检查并完善合同信息`。
- 商机已有合同时提供“查看现有合同”。
- 商机状态不合格时提示返回商机详情。
- 客户不一致时禁止手工修正客户字段。

### 9.3 保存成功

新增模式：

- 生成合同编号。
- 合同进入草稿。
- 建立有效合同绑定。
- 商机联动进入合同阶段。
- Toast `合同创建成功`。
- 跳转合同详情页。

编辑模式：

- 保存合同信息。
- 状态保持草稿。
- 不重复推进商机。
- Toast `合同信息已更新`。
- 返回合同详情页。

### 9.4 保存失败

- 保留所有表单输入。
- 不创建孤立合同。
- 不改变商机阶段。
- 并发冲突提示 `合同已被他人更新`。
- 网络结果不确定时按请求键查询结果。
- 失败 Toast 展示可理解原因。
