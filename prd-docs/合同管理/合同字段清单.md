# 合同字段清单

> **版本**：V1.0 | 2026-07-18

## 一、基础字段

| 字段名称 | 字段来源 | 取值说明 | 必填性 | 列表展示 | 可筛选 | 详情展示 |
|----------|---------|---------|:------:|:------:|:------:|:------:|
| 合同编号 | 系统生成 | `CT{YYYYMMDD}-{4位序号}` | 系统自动 | 是(160px) | 是(Input) | 是 |
| 合同名称 | 用户填写 | 文本,上限100字符 | 必填 | 是(200px) | 是(Input) | 是 |
| 关联客户 | 系统带出 | 从商机继承 | 必填 | 是(160px) | 是(Select) | 是 |
| 关联商机 | 系统带出 | 商机编号 | 必填 | 是(160px) | 否 | 是 |
| 合同金额 | 用户填写 | 元,2位小数 | 必填 | 是(140px) | 否 | 是 |
| 合同状态 | 系统生成 | DRAFT/PENDING_SIGN/SIGNED/ARCHIVED/VOIDED | 系统自动 | 是(100px) | 是(Select) | 是 |
| 签署日期 | 系统生成 | `YYYY-MM-DD` | 系统自动 | 是(140px) | 是(DateRange) | 是 |
