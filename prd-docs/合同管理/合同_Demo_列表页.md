# 合同_Demo_列表页

> **版本**：V2.0 | 2026-07-18
> **页面类型**：PC Web 列表页
> **前置文档**：《合同主PRD》《合同字段清单》
> **字段约束**：字段定义、状态口径与 Tag 配色引用《合同字段清单》，本文不重复定义。

---

## 1. 页面概述

| 项目 | 内容 |
|------|------|
| 页面名称 | 合同管理 |
| 页面路径 | `/contracts` |
| 页面目标 | 查询合同、追踪签署进度并执行状态允许的动作 |
| 主要用户 | 销售、销售主管、系统管理员、只读审计角色 |
| 页面骨架 | 页头 → 状态 Tab → 查询区 → 工具条 → 表格 → 分页 |
| 默认排序 | 创建时间倒序 |
| 每页条数 | 默认 20 条，可切换 20 / 50 / 100 |
| 详情入口 | 合同编号链接 |

页面原则：

- 状态只通过动作按钮或签署事件变化。
- 列表不提供状态下拉直接修改。
- 已签署与已归档合同不可作废。
- 不提供删除合同。
- 关联客户和商机使用只读链接。
- 权限与《合同主PRD》一致。
- 表格最小宽度 1480px。

## 2. 状态 Tab 与后台计数

### 2.1 Tab 规格

| Tab | 过滤语义 | 计数来源 | 点击行为 |
|-----|----------|----------|----------|
| 全部 | 当前权限范围内全部合同 | 后台聚合 | 清除状态过滤，保留其他条件 |
| 草稿 | 草稿合同 | 后台聚合 | 筛选草稿并回到第一页 |
| 待签署 | 待签署合同 | 后台聚合 | 筛选待签署并回到第一页 |
| 已签署 | 已签署未归档合同 | 后台聚合 | 筛选已签署并回到第一页 |
| 已归档 | 已归档合同 | 后台聚合 | 筛选已归档并回到第一页 |
| 已作废 | 已作废合同 | 后台聚合 | 筛选已作废并回到第一页 |

### 2.2 计数规则

- 格式为 `Tab名称 (计数)`。
- 计数按当前用户数据范围聚合。
- 计数受查询区其他条件影响。
- 计数不受分页影响。
- 提交签署后草稿减一、待签署加一。
- 签署完成后待签署减一、已签署加一。
- 归档后已签署减一、已归档加一。
- 作废后来源状态减一、已作废加一。
- 计数失败显示 `--`，不阻断表格。
- 从详情返回时恢复 Tab 和页码。

## 3. 查询区

### 3.1 布局

- 默认两行自适应布局。
- 标签位于控件上方。
- “查询”为主按钮。
- “重置”为次按钮。
- 查询后回到第一页。
- 查询条件写入 URL。

### 3.2 控件详细规格

| 字段名 | 控件 | 宽度 | Placeholder | 规格 | 查询行为 |
|--------|------|------|-------------|------|----------|
| 合同编号 | Input | 200px | `请输入合同编号` | 支持粘贴、清空 | 精确匹配优先 |
| 合同名称 | Input | 220px | `请输入合同名称` | 去除首尾空格 | 模糊匹配 |
| 关联客户 | SelectWithSearch | 220px | `请选择关联客户` | 远程搜索可见客户 | 按客户标识过滤 |
| 关联商机 | SelectWithSearch | 220px | `请选择关联商机` | 远程搜索可见商机 | 按商机标识过滤 |
| 合同状态 | Select | 180px | `请选择合同状态` | 选项引用字段清单 | 与 Tab 条件联动 |
| 签署日期 | DateRangePicker | 260px | `开始日期 - 结束日期` | 含首尾日期 | 范围过滤 |
| 创建人 | SelectWithSearch | 180px | `请选择创建人` | 主管和管理员可见 | 按用户标识过滤 |
| 创建时间 | DateRangePicker | 260px | `开始日期 - 结束日期` | 最大跨度 1 年 | 范围过滤 |

### 3.3 查询反馈

- 合同名称输入回车触发查询。
- 查询中禁用查询按钮。
- 查询失败保留旧结果。
- Toast `合同查询失败，请重试`。
- 重置清空条件并回到“全部”。
- 无结果显示 `未找到符合条件的合同`。
- 快速查询取消旧请求。

## 4. 工具条

| 控件 | 位置 | 样式 | 权限 | 行为 |
|------|------|------|------|------|
| 新建合同 | 左侧 | Plus 图标 + 主按钮 | 有新增权限 | 进入新增页 |
| 刷新 | 左侧 | RefreshCw 图标按钮 | 全部 | 保留条件刷新 |
| 导出 | 右侧 | Download 图标 + 文案 | 按权限 | 导出当前查询范围 |
| 列设置 | 右侧 | Columns3 图标按钮 | 全部 | 配置列显隐 |

工具条约束：

- 不展示删除或批量作废。
- 图标按钮必须有 Tooltip。
- 导出遵循数据权限。
- 导出不限当前页。
- 导出进行中防重复。
- 新建页仍需校验可关联商机。

## 5. 表格规格

### 5.1 表格行为

- 表头吸顶。
- 合同编号列左固定。
- 操作列右固定。
- 行高 56px。
- 不支持行选择和批量操作。
- 不支持整行点击。
- 合同编号点击进入详情。
- 空值统一显示 `--`。
- 排序切换后回到第一页。

### 5.2 列定义

| 列名 | 宽度 | 固定 | 对齐 | 展示规则 |
|------|------|------|------|----------|
| 合同编号 | 168px | 左 | 左 | 链接，等宽文本 |
| 合同名称 | 220px | 否 | 左 | 两行省略，Tooltip 全文 |
| 关联客户 | 180px | 否 | 左 | 链接进入客户详情 |
| 关联商机 | 180px | 否 | 左 | 链接进入商机详情 |
| 合同金额 | 148px | 否 | 右 | 格式化金额 |
| 合同状态 | 120px | 否 | 中 | Tag 引用字段清单 |
| 签署日期 | 144px | 否 | 中 | 无值显示 `--` |
| 创建人 | 112px | 否 | 左 | 单行省略 |
| 创建时间 | 176px | 否 | 中 | 完整时间 |
| 最后修改时间 | 176px | 否 | 中 | 完整时间 |
| 联动结果 | 148px | 否 | 中 | 商机/ERP 联动摘要 |
| 操作 | 220px | 右 | 左 | 动态操作，超出进入 More |

### 5.3 联动结果展示

| 合同场景 | 文案 | 视觉 |
|----------|------|------|
| 尚未触发 | `--` | 常规 |
| 商机联动中 | `商机同步中` | Loader 图标 |
| 商机赢单成功 | `商机已赢单` | Check 图标 |
| ERP 订单成功 | `ERP订单已生成` | Check 图标 |
| 联动失败 | `联动异常` | AlertTriangle + Tooltip |
| 合同作废回退 | `商机已回退` | Undo2 图标 |

联动结果是展示元数据，不作为合同业务字段编辑。

## 6. 行内操作动态矩阵

| 合同状态 | 主操作 | 次操作 | More 菜单 |
|----------|--------|--------|-----------|
| 草稿 | 提交签署 | 查看、编辑 | 作废 |
| 待签署 | 模拟签署完成 | 查看 | 作废 |
| 已签署 | 归档 | 查看 | 查看商机、查看 ERP 结果 |
| 已归档 | 查看 | 无 | 查看商机、查看 ERP 结果 |
| 已作废 | 查看 | 无 | 查看历史商机 |

动态规则：

- 动作先服从合同状态，再服从角色权限。
- 无权限动作不渲染。
- 不可用动作不使用 disabled 展示。
- 已签署与已归档不展示作废。
- 任一状态不展示删除。
- 模拟签署完成仅主管和管理员展示。
- 联动失败时主管或管理员在 More 中看到“重试联动”。
- ERP 下推失败时主管或管理员看到“重试 ERP”。

## 8. 并发与失败反馈

| 场景 | 页面处理 |
|------|----------|
| 状态已被他人改变 | 刷新当前行并 Toast `合同状态已变化` |
| 同一商机已有合同 | 阻断创建并提供“查看现有合同” |
| 签署回调重复 | 返回已有结果，不重复刷新计数 |
| 商机赢单失败 | 合同显示已签署，联动结果显示异常 |
| ERP 下推失败 | 合同状态不回退，展示重试入口 |
| 作废回退失败 | 当前行保持原状态，显示事务失败文案 |
| 查询子数据失败 | 表格主数据保留，联动列显示 `--` |

## 9. 分页、加载与空态

### 9.1 分页

- 默认每页 20 条。
- 支持 20、50、100 条。
- 显示总条数。
- 切页滚动到表格顶部。
- 改变每页条数回到第一页。
- URL 保存页码与每页条数。

### 9.2 加载

- 初次加载显示 10 行骨架。
- 刷新时保留表头和旧数据。
- 行内动作只锁定当前行。
- Tab 计数独立加载。
- 快速查询取消旧请求。

### 9.3 空态

| 场景 | 文案 | 动作 |
|------|------|------|
| 无合同 | `暂无合同` | 有权限时新建合同 |
| 查询无结果 | `未找到符合条件的合同` | 重置筛选 |
| 某状态为空 | `当前状态暂无合同` | 返回全部 |
| 服务异常 | `合同数据暂不可用` | 重试 |
