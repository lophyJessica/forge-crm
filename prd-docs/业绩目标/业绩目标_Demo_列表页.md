# 业绩目标_Demo_列表页

> **版本**：V2.0 | 2026-07-18
> **页面类型**：PC Web 列表页
> **前置文档**：《业绩目标主PRD》《业绩目标字段清单》
> **字段约束**：字段定义、状态和进度配色引用《业绩目标字段清单》，本文不重复定义。

---

## 1. 页面概述

| 项目 | 内容 |
|------|------|
| 页面名称 | 业绩目标 |
| 页面路径 | `/performance-targets` |
| 页面目标 | 按月份查看销售三项目标、实时进度与结算结果 |
| 主要用户 | 销售、销售主管、系统管理员、只读审计角色 |
| 页面骨架 | 页头 → 月份导航 → 状态 Tab → 查询区 → 工具条 → 表格 → 分页 |
| 默认月份 | 当前月 |
| 默认排序 | 销售ID升序 |
| 每页条数 | 默认 20 条，可切换 20 / 50 / 100 |

页面原则：

- 一行代表一个销售某个月的一条目标。
- 使用销售ID作为稳定归属标识。
- 姓名仅展示，不作为查询唯一键。
- 三项进度并列展示，避免只看金额。
- 进行中数据可刷新。
- 终态数据读取结算快照。
- 不提供状态直接修改。
- 不提供删除。
- 表格最小宽度 1840px。

## 2. 月份导航

### 2.1 控件

| 控件 | 规格 | 行为 |
|------|------|------|
| 上一月 | ChevronLeft 图标按钮 | 切换到前一个月 |
| 当前月份 | MonthPicker，宽 160px | 选择目标月份 |
| 下一月 | ChevronRight 图标按钮 | 切换到后一个月 |
| 回到本月 | 次按钮 | 切换当前月 |

### 2.2 月份行为

- 月份写入 URL。
- 切换月份后回到第一页。
- 切换月份清空状态 Tab 以外的冲突条件。
- 当前月显示 `进行中月份` 弱提示。
- 历史月显示 `已结算月份` 或异常提示。
- 未来月显示 `计划月份`。
- 下一月按钮允许查看未来已创建目标。
- MonthPicker 不允许无值提交查询。
- 月份切换取消上一请求。

### 2.3 时区说明

- 月份边界统一按 Asia/Shanghai。
- Tooltip 展示 `结算时区：Asia/Shanghai`。
- 浏览器本地时区不改变业务月份。
- 历史结算使用后端返回的固定截止时间。

## 3. 状态 Tab 与后台计数

### 3.1 Tab 规格

| Tab | 过滤语义 | 计数来源 | 点击行为 |
|-----|----------|----------|----------|
| 全部 | 当前月份与权限范围内全部目标 | 后台聚合 | 清除状态条件 |
| 进行中 | 尚未完成月末结算 | 后台聚合 | 筛选并回到第一页 |
| 已达成 | 结算快照判定达成 | 后台聚合 | 筛选并回到第一页 |
| 未达成 | 结算快照判定未达成 | 后台聚合 | 筛选并回到第一页 |
| 结算异常 | 结算失败或待补偿的技术元数据 | 后台聚合 | 筛选进行中且结算异常记录 |

### 3.2 计数规则

- 格式为 `名称 (计数)`。
- 计数按月份和数据权限聚合。
- 计数受销售条件影响。
- 计数不受分页影响。
- 创建目标后全部与进行中加一。
- 结算完成后进行中转移至对应终态。
- 结算失败只进入技术异常聚合，不伪造业务终态。
- 计数失败显示 `--`。
- 计数失败不阻断表格。
- 从编辑返回时恢复月份和 Tab。

## 4. 查询区

### 4.1 布局

- 查询区单行自适应，空间不足换行。
- 控件标签位于上方。
- “查询”为主按钮。
- “重置”为次按钮。
- 查询后回第一页。
- 条件写入 URL。

### 4.2 控件详细规格

| 条件/字段 | 控件 | 宽度 | Placeholder | 规格 | 查询行为 |
|-----------|------|------|-------------|------|----------|
| 目标编号 | Input | 220px | `请输入目标编号` | 支持粘贴、清空 | 精确匹配优先 |
| 销售 | SelectWithSearch | 220px | `请选择销售` | 选项展示姓名与销售ID | 按销售ID过滤 |
| 目标状态 | Select | 180px | `请选择目标状态` | 选项引用字段清单 | 与 Tab 联动 |
| 达成情况 | Select | 180px | `请选择达成情况` | 三项全部达标 / 存在未达标 | 后台计算过滤 |

### 4.3 销售选择器

- 远程搜索姓名或销售ID。
- 选项第一行展示姓名。
- 第二行展示销售ID和团队。
- 提交查询只发送销售ID。
- 同名销售通过ID区分。
- 未映射用户不出现在候选中。
- 销售本人默认锁定为自己，主管可选择团队成员。

### 4.4 查询反馈

- 目标编号输入按回车触发查询。
- 查询中禁用按钮。
- 查询失败保留旧数据。
- Toast `业绩目标查询失败，请重试`。
- 重置保留当前月份并回到“全部”。
- 无结果显示 `未找到符合条件的业绩目标`。
- 快速查询取消旧请求。

|------|------|------|------|------|

| 刷新进度 | 左侧 | RefreshCw 图标 + 文案 | 进行中目标可用 | 重新查询实时聚合 |

- 不展示批量修改状态。

## 6. 表格规格

### 6.1 表格行为

- 表头吸顶。
- 目标编号列左固定。
- 操作列右固定。
- 行高 72px，保证三项进度可读。
- 不支持行选择和批量操作。
- 不支持整行点击。
- 空值显示 `--`。
- 终态行使用结算快照，不实时刷新。
- 进行中刷新保留表格宽度。

### 6.2 列定义

| 列名 | 宽度 | 固定 | 对齐 | 展示规则 |
|------|------|------|------|----------|
| 目标编号 | 200px | 左 | 左 | 等宽文本，可复制 |
| 销售 | 160px | 否 | 左 | 姓名 + 销售ID |
| 目标月份 | 120px | 否 | 中 | 月份文本 |
| 线索目标数 | 120px | 否 | 右 | 数字 |
| 当前线索转化数 | 148px | 否 | 右 | 当前值 / 目标值 |
| 线索达成率 | 180px | 否 | 左 | 进度条 + 百分比 |
| 商机目标数 | 120px | 否 | 右 | 数字 |
| 当前商机数 | 128px | 否 | 右 | 当前值 / 目标值 |
| 商机达成率 | 180px | 否 | 左 | 进度条 + 百分比 |
| 赢单目标金额 | 160px | 否 | 右 | 格式化金额 |
| 当前赢单金额 | 160px | 否 | 右 | 当前值 / 目标值摘要 |
| 赢单达成率 | 180px | 否 | 左 | 进度条 + 百分比 |
| 目标状态 | 128px | 否 | 中 | Tag 引用字段清单 |
| 数据时间 | 176px | 否 | 中 | 实时聚合或结算时间 |
| 操作 | 176px | 右 | 左 | 动态操作 |

### 6.3 进度条规格

- 宽度固定 140px。
- 高度固定 8px。
- 颜色引用《业绩目标字段清单》。
- 百分比显示在进度条右侧。
- 超过 100% 时进度条填满，文本显示真实百分比。
- 目标值异常时不做除零，显示 `--` 并提示数据错误。
- 加载时使用同宽骨架。
- 颜色不是唯一表达，必须显示数值。

### 6.4 数据时间

| 目标场景 | 文案 |
|----------|------|
| 进行中 | `更新于 {时间}` |
| 结算中 | `结算中` + Loader |
| 已结算 | `结算于 {时间}` |
| 聚合延迟 | `数据可能延迟` + Tooltip |
| 结算失败 | `结算异常` + Tooltip |

## 7. 行内操作动态矩阵

| 目标状态/元数据 | 主操作 | 次操作 | More |
|-----------------|--------|--------|------|
| 进行中正常 | 查看明细 | 编辑（按权限） | 刷新进度 |
| 进行中结算中 | 查看明细 | 无 | 无 |
| 进行中结算失败 | 查看明细 | 重试结算（按权限） | 查看错误摘要 |
| 已达成 | 查看结算 | 无 | 查看差异审计 |
| 未达成 | 查看结算 | 无 | 查看差异审计 |

动态规则：

- 销售没有编辑入口。
- 主管只编辑团队进行中目标。
- 管理员可编辑全部进行中目标。
- 终态不展示编辑。
- 任一状态不展示删除。
- 不可用动作不渲染。
- More 使用 Ellipsis 图标并有 Tooltip。

## 8. 明细抽屉

### 8.1 打开方式

- 点击“查看明细”或三项当前值。
- 从右侧打开 760px 抽屉。
- 默认定位到用户点击的指标 Tab。
- 抽屉不改变列表筛选。

### 8.2 指标 Tab

| Tab | 数据来源 | 关键时间 |
|-----|----------|----------|
| 线索转化 | 线索转客户事件 | 转化时间 |
| 商机创建 | 商机创建事件 | 创建时间 |
| 赢单金额 | 商机赢单事件 | 赢单时间 |

### 8.3 明细列

共同列：

- 来源对象编号。
- 事件时间。
- 事件销售ID。
- 当前展示姓名。
- 计入值。
- 是否计入。
- 去重事件标识摘要。

明细规则：

- 进行中读取实时事件。
- 已结算读取结算明细快照。
- 已结算后晚到事件不插入历史快照。
- 差异在独立审计视图展示。
- 明细权限不越过来源对象权限。
- 无权明细以聚合摘要替代。

## 9. 编辑与结算弹窗

### 9.1 月内编辑确认

| 项目 | 规格 |
|------|------|
| 触发 | 当前月份进行中目标点击编辑 |
| 标题 | `调整目标` |
| 内容 | `当前月份已开始，调整目标会立即改变达成率并记录审计。` |
| 取消 | `取消` |
| 确认 | `继续编辑` |

未来月份目标直接进入编辑页，不弹此确认。

### 9.2 重试结算

| 项目 | 规格 |
|------|------|
| 标题 | `重试结算` |
| 内容 | `将沿用原月末截止时间和幂等键重新结算。` |
| 取消 | `取消` |
| 确认 | `确认重试` |
| 成功 Toast | `结算任务已重新提交` |
| 失败 Toast | `结算重试失败，请稍后再试` |

### 9.3 导出

- 常规数据量直接创建导出任务。
- 大数据量弹后台导出确认。
- 导出进行中禁止重复。
- 已结算月导出快照值。
- 当前月导出实时聚合值并带数据时间。

## 10. 防漂移展示

### 10.1 已结算行

- 显示 Lock 图标。
- Tooltip：`该目标已结算，当前展示锁定快照。`。
- 数据时间使用结算时间。
- 刷新按钮不改变数值。
- 姓名可展示最新名称，但销售ID保持不变。
- 状态与三项当前值只读。

### 10.2 差异审计

| 差异场景 | 展示 |
|----------|------|
| 晚到线索转化 | 事件时间与到达时间 |
| 晚到商机创建 | 事件时间与到达时间 |
| 赢单金额修正 | 快照金额与源数据当前金额 |
| 销售改名 | 原显示名与当前显示名 |
| 销售转组 | 事件时团队与当前团队 |

差异只读展示，不提供“应用到历史”按钮。

## 11. 分页、加载与空态

### 11.1 分页

- 默认 20 条。
- 支持 20、50、100 条。
- 显示总条数。
- 切页回表格顶部。
- 改变每页条数回第一页。
- URL 保存分页状态。

### 11.2 加载

- 初次加载显示 8 行固定骨架。
- 进行中进度刷新覆盖局部加载层。
- 终态行不显示实时刷新动画。
- Tab 计数独立加载。
- 月份切换取消旧请求。

### 11.3 空态

| 场景 | 文案 | 动作 |
|------|------|------|
| 当前月无目标 | `当前月份暂无业绩目标` | 有权限时新建目标 |
| 历史月无目标 | `该月份没有业绩目标` | 切换月份 |
| 查询无结果 | `未找到符合条件的业绩目标` | 重置筛选 |
| 服务异常 | `业绩目标数据暂不可用` | 重试 |
