# 业绩目标字段清单

> **版本**：V1.0 | 2026-07-18

## 基础字段

| 字段名称 | 字段来源 | 必填性 | 列表展示 |
|----------|---------|:------:|:------:|
| 目标编号 | 系统生成 `TGT{YYYYMM}-{销售ID}` | 系统自动 | 是 |
| 销售 | 用户填写 | 必填 | 是 |
| 目标月份 | 用户填写 `YYYY-MM` | 必填 | 是 |
| 线索目标数 | 用户填写 | 必填 | 是 |
| 商机目标数 | 用户填写 | 必填 | 是 |
| 赢单目标金额 | 用户填写 | 必填 | 是 |
| 当前线索转化数 | 系统计算 | 系统自动 | 是 |
| 当前商机数 | 系统计算 | 系统自动 | 是 |
| 当前赢单金额 | 系统计算 | 系统自动 | 是 |
| 目标状态 | 系统生成 ACTIVE/ACHIEVED/UNACHIEVED | 系统自动 | 是 |

## 进度条配色

| 进度 | 颜色 |
|------|------|
| ≥100% | 绿色 |
| 50-99% | 黄色 |
| <50% | 红色 |
