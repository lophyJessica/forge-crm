# 业绩目标_Demo_新增编辑页

> **版本**：V2.0 | 2026-07-18
> **页面类型**：PC Web 新增页 / 编辑页
> **前置文档**：《业绩目标主PRD》《业绩目标字段清单》
> **字段约束**：字段定义、必填性、格式和限制以《业绩目标字段清单》为唯一权威源。

---

## 1. 页面概述

| 项目 | 新增模式 | 编辑模式 |
|------|----------|----------|
| 页面名称 | 新建业绩目标 | 编辑业绩目标 |
| 页面路径 | `/performance-targets/new` | `/performance-targets/{id}/edit` |
| 进入方式 | 列表“新建目标” | 列表“编辑” |
| 页面目标 | 为销售创建月度三项目标 | 调整尚未结算的三项目标值 |
| 可用状态 | 新建 | 仅进行中 |
| 页面骨架 | 页头 + 目标表单 + 编号预览 + 固定底栏 | 页头 + 状态摘要 + 目标表单 + 当前进度只读 + 固定底栏 |

页面原则：

- 使用销售ID而不是姓名建立目标归属。
- 销售与月份创建后不可更换。
- 三项目标值由主管维护。
- 三项当前进度由系统聚合。
- 目标状态由系统结算。
- 表单不提供状态编辑。
- 已结算目标不可进入编辑页。
- 不提供删除。
- 表单最大宽度 920px。

## 2. 路由与初始化

### 2.1 新增模式

- 校验目标新增权限。
- 从列表进入时预填列表当前月份。
- 销售默认为空。
- 三项目标值为空。
- 当前进度不展示或显示初始只读摘要。
- 状态不展示。
- 目标编号显示预览占位。
- 不在前端最终生成编号。

### 2.2 编辑模式

- 根据路径标识加载目标。
- 校验查看权限。
- 校验编辑权限。
- 校验目标仍为进行中。
- 展示目标编号与状态只读摘要。
- 销售和月份只读。
- 三项目标值可编辑。
- 三项当前进度只读展示。
- 已结算目标直接跳转列表。
- Toast `已结算目标不可修改`。

### 2.3 路由参数

| 参数 | 来源 | 用途 | 异常处理 |
|------|------|------|----------|
| `id` | 编辑路径 | 查询目标 | 不存在进入 404 |
| `month` | 列表月份 | 预填目标月份 | 非法值回当前月 |
| `from` | 来源页 | 返回上下文 | 非法值回目标列表 |

### 2.4 进入页面校验

- 销售本人无新增或编辑权限时返回 403。
- 主管只能为团队成员创建目标。
- 管理员可选择全部已映射销售。
- 未映射销售ID的用户不进入候选。
- 编辑页面若结算锁已生效，立即跳转只读列表或详情上下文。

## 5. 当前进度只读区

### 5.1 编辑模式展示

| 指标 | 展示 | 说明 |
|------|------|------|
| 线索进度 | 当前值 / 当前目标 + 进度条 | 当前目标变化时仅预览新比例 |
| 商机进度 | 当前值 / 当前目标 + 进度条 | 当前值不可编辑 |
| 赢单进度 | 当前金额 / 当前目标 + 进度条 | 当前金额不可编辑 |

### 5.2 预览规则

- 进度条颜色引用字段清单。
- 目标值变化后即时预览。
- 当前值不随表单输入变化。
- 超过 100% 显示真实百分比。
- 预览不改变后台状态。
- 提前三项达标仍显示进行中提示。
- 保存失败时恢复后台目标比例。

### 5.3 数据时间

- 展示当前进度聚合时间。
- 聚合延迟时显示提示。
- 编辑页面不提供修正当前值。
- 如进度有误，引导查看事件明细。
- 不允许用调整目标值伪造源事件修正。

## 6. 字段联动规则

| 编号 | 触发 | 目标 | 行为 |
|------|------|------|------|
| L01 | 选择销售 | 目标编号预览 | 使用销售ID更新预览 |
| L02 | 选择目标月份 | 目标编号预览 | 使用月份更新预览 |
| L03 | 销售或月份变化 | 唯一性校验 | 远程预检是否已有目标 |
| L04 | 三项目标变化 | 三项达成率 | 仅前端预览 |
| L05 | 编辑目标保存 | 审计日志 | 记录修改前后值 |
| L06 | 当前月目标编辑 | 二次确认 | 保存前确认目标调整影响 |
| L07 | 保存成功 | 列表进度 | 使用新目标重新展示实时比例 |
| L08 | 保存失败 | 当前目标 | 后台值不变化，表单输入保留 |

联动约束：

- 姓名变化不更新目标编号。
- 销售ID与月份创建后不可修改。
- 当前进度不可由表单联动覆盖。
- 状态不可由预览结果改变。
- 提前三项达标不提前结算。

## 7. 唯一性预检

### 7.1 触发

- 销售与月份都已选择。
- 300ms 防抖后请求后台。
- 输入未完成时不请求。
- 保存前仍执行最终唯一约束。

### 7.2 结果

| 结果 | 页面表现 |
|------|----------|
| 可创建 | 绿色提示 `该销售本月尚未设置目标` |
| 已存在 | 红色提示 + 现有目标编号链接 |
| 销售ID未映射 | 错误提示并阻断 |
| 查询失败 | 弱提示，保存前再次校验 |

### 7.3 已存在处理

- 保存按钮保持可点击但提交会阻断，或前端根据设计禁用并展示原因。
- 提供“查看现有目标”。
- 不提供覆盖创建。
- 不提供删除现有目标。
- 不通过更改姓名绕过唯一性。

## 8. 新增与编辑模式差异

| 能力 | 新增 | 进行中编辑 | 已结算直接访问 |
|------|:----:|:----------:|:--------------:|
| 销售 | 可选择 | 只读 | 不允许进入 |
| 目标月份 | 可选择/预填 | 只读 | 不允许进入 |
| 三项目标值 | 可编辑 | 可编辑 | 不允许进入 |
| 三项当前值 | 初始摘要 | 只读实时值 | 详情快照只读 |
| 目标编号 | 预览 | 只读正式值 | 详情只读 |
| 目标状态 | 不展示 | 只读 | 详情只读 |
| 保存 | 创建目标 | 保存修改 | 不展示 |

编辑规则：

- 当前月编辑进入前弹一次确认。
- 未来月编辑不弹月内确认。
- 历史已结算不可编辑。
- 销售和月份不可更换。
- 如创建归属错误，不支持删除或改归属，需由管理员按未来校正流程处理。
- 编辑不改变目标状态。

## 11. 保存交互规范

### 11.1 校验顺序

1. 当前用户权限。
2. 编辑模式目标状态。
3. 销售ID映射。
4. 字段清单校验。
5. 销售+月份唯一性。
6. 当前月调整确认。
7. 版本并发校验。
8. 保存目标与审计日志。

### 11.2 校验失败

- 所有错误字段同时标红。
- 页面滚动并聚焦第一个错误。
- Toast `请检查并完善目标设置`。
- 销售ID未映射时给出管理员处理提示。
- 重复目标提供现有目标链接。
- 数值错误定位到具体目标控件。

### 11.3 保存成功

新增：

- 后台生成正式目标编号。
- 目标进入进行中。
- 当前进度按当月事件聚合。
- Toast `业绩目标创建成功`。
- 返回目标列表并定位当前月份。

编辑：

- 保存新目标值。
- 记录修改前后审计。
- 状态仍为进行中。
- 列表按新目标计算显示比例。
- Toast `业绩目标已更新`。

### 11.4 保存失败

- 保留表单输入。
- 后台目标不变化。
- 当前进度不变化。
- 并发结算时提示 `目标已进入结算，无法继续修改`。
- 并发编辑时提示 `目标已被他人更新`。
- 网络结果不确定时按请求键查询最终结果。

### 14.1 新增当前月目标

- 月份预填当前月。
- 销售为空。
- 选择张三后使用 `S001` 预览编号。
- 唯一性预检通过。
- 三项目标为空。
- 保存后进入进行中。

### 14.2 新增重复目标

- 选择李四与已有目标月份。
- 唯一性预检返回已存在。
- 展示现有目标链接。
- 保存阻断。

### 14.3 未映射销售

- 搜索结果不展示未映射用户。
- 通过旧参数进入时后台阻断。
- 提示联系管理员完成销售ID映射。

### 14.4 当前月编辑

- 销售和月份只读。
- 展示三项实时进度。
- 调整任一目标后进度预览变化。
- 保存前展示修改前后对比。
- 保存后状态仍进行中。

### 14.5 未来月编辑

- 销售与月份只读。
- 三项目标可编辑。
- 不显示实时进度或显示初始零值。
- 不弹月内调整确认。

### 14.6 结算并发

- 编辑页打开后模拟月末结算锁。
- 保存返回结算锁冲突。
- 表单不覆盖结算快照。
- 跳转列表显示终态。

## 15. 键盘与可访问性

- Tab 顺序与视觉顺序一致。
- 所有控件有可读 Label。
- 错误文案关联控件。
- Select 支持键盘搜索。
- MonthPicker 支持键盘输入。
- NumberInput 支持步进器但可直接输入。
- 进度条同时显示数值。
- 保存结果通过 aria-live 播报。
- 固定底栏不遮挡表单末尾。
