# 客户_Demo_列表页

> **版本**：V2.0 | 2026-07-18
> **页面类型**：PC Web 列表页
> **前置文档**：《客户主PRD》《客户字段清单》
> **SSOT 声明**：客户正式档案来自 ERP，页面不提供新增、编辑或删除客户能力。
> **字段约束**：字段定义与展示口径引用《客户字段清单》。ERP 同步标识是技术元数据提示，不是客户业务字段。

---

## 1. 页面概述

| 项目 | 内容 |
|------|------|
| 页面名称 | 客户管理 |
| 页面路径 | `/accounts` |
| 页面目标 | 查询 ERP 客户快照，并查看 CRM 关系聚合与流失风险 |
| 主要用户 | 销售、销售主管、系统管理员、只读审计角色 |
| 页面骨架 | 页头 → 风险 Tab → 查询区 → 工具条 → 表格 → 分页 |
| 默认排序 | 引用《客户字段清单》展示规则 |
| 每页条数 | 默认 20 条，可切换 20 / 50 / 100 |
| 详情入口 | 客户编号或公司名称链接 |

页面原则：

- 首屏直接展示客户工作列表，不做介绍页。
- 页面任何位置都不得出现“新增客户”按钮。
- 不展示“编辑客户”或“删除客户”。
- 基础档案异常时引导用户在 ERP 核对。
- CRM 只提供新建商机、跟进等关联业务动作。
- 数据权限与《客户主PRD》一致。
- 表格最小宽度 1680px，窄屏横向滚动。

## 2. 风险 Tab 与计数

### 2.1 Tab 规格

| Tab | 过滤语义 | 计数来源 | 点击行为 |
|-----|----------|----------|----------|
| 全部客户 | 当前权限范围内全部客户快照 | 后台聚合 | 保留查询条件，清除风险过滤 |
| 高风险 | 最新成功结果为高风险 | 后台聚合 | 应用高风险过滤并回到第一页 |
| 中风险 | 最新成功结果为中风险 | 后台聚合 | 应用中风险过滤并回到第一页 |
| 低风险 | 最新成功结果为低风险 | 后台聚合 | 应用低风险过滤并回到第一页 |
| 待计算 | 尚无成功风险结果 | 后台聚合 | 展示待计算与失败不可用数据 |

### 2.2 计数规则

- 格式为 `Tab名称 (计数)`。
- 计数按当前用户数据范围聚合。
- 计数受其他查询条件影响。
- 计数不受分页影响。
- 风险计算完成后刷新各 Tab 计数。
- ERP 快照新增后刷新“全部客户”计数。
- 计数接口失败显示 `--`。
- 计数失败不阻断客户表格查询。
- 快速切换 Tab 时取消旧请求。
- 从详情返回时恢复原 Tab。

## 3. 查询区

### 3.1 布局

- 默认两行自适应布局。
- 单个控件标签在上方。
- 第一行放高频客户筛选。
- 第二行放风险、时间和同步筛选。
- “查询”为主按钮。
- “重置”为次按钮。
- 查询后页码重置为 1。
- 查询条件写入 URL。

### 3.2 控件详细规格

| 字段/条件 | 控件 | 宽度 | Placeholder | 规格 | 查询行为 |
|-----------|------|------|-------------|------|----------|
| 客户编号 | Input | 200px | `请输入客户编号` | 支持粘贴和清空 | 精确匹配优先 |
| 公司名称 | Input | 220px | `请输入公司名称` | 去除首尾空格 | 模糊匹配 |
| 手机号 | Input | 180px | `请输入手机号` | 格式校验引用字段清单 | 精确匹配 |
| 所属行业 | Select | 180px | `请选择所属行业` | 选项引用字段清单；支持清空 | 单选过滤 |
| 所在地区 | Cascader | 220px | `请选择所在地区` | 支持按省或市选择 | 层级过滤 |
| 客户等级 | Select | 160px | `请选择客户等级` | 选项引用字段清单；支持清空 | 单选过滤 |
| AI流失风险 | Select | 180px | `请选择流失风险` | 选项引用字段清单；支持清空 | 与风险 Tab 取交集 |
| 最近跟进时间 | DateRangePicker | 260px | `开始日期 - 结束日期` | 首尾日期均包含 | 范围过滤 |
| ERP同步标识 | Select | 180px | `请选择同步状态` | 技术元数据选项；不写入客户字段 | 按同步任务结果过滤 |

### 3.3 查询交互

- 公司名称输入按回车触发查询。
- 手机号格式错误时不发请求并就地提示。
- 风险 Tab 与 AI流失风险条件冲突时以后操作条件为准。
- 查询中按钮展示加载图标。
- 查询失败保留上一批结果。
- 查询失败 Toast `客户查询失败，请重试`。
- 重置清空全部条件并回到“全部客户”。
- 重置不改变每页条数。

## 5. ERP 同步标识

### 5.1 定位

ERP 同步标识用于告诉用户 CRM 当前展示的快照是否为最近一次成功同步结果。它是同步任务元数据，不是可编辑客户字段，也不替代 ERP 客户状态。

### 5.2 展示状态

| 标识文案 | 图标 | 视觉 | 说明 |
|----------|------|------|------|
| 已同步 | CheckCircle2 | 绿色 | 最近一次同步成功 |
| 同步中 | LoaderCircle | 蓝色旋转 | 正在处理新版本 |
| 同步异常 | AlertTriangle | 红色 | 最近一次处理失败，仍展示旧快照 |
| ERP已停用 | CircleSlash2 | 灰色 | ERP 已停用，CRM 历史只读 |

### 5.3 交互

- 悬浮展示最后成功同步时间。
- 同步异常 Tooltip 展示用户可理解的错误摘要。
- 管理员在同步异常行的 More 菜单可执行“重试同步”。
- 销售仅可查看提示，不可重试。
- 点击“ERP已停用”跳转详情历史视图。
- 不允许用户通过列表修改同步标识。
- 同步标识加载中保持列宽稳定。

## 7. 行内操作

### 7.1 动作矩阵

| 客户快照场景 | 主操作 | More 菜单 |
|--------------|--------|-----------|
| 正常可用 | 查看 | 新建商机、添加跟进、创建拜访 |
| 高流失风险 | 查看 | 添加跟进、新建商机、创建拜访 |
| 同步中 | 查看 | 无 |
| 同步异常 | 查看 | 管理员显示重试同步 |
| ERP已停用 | 查看 | 无 |

### 7.2 动态规则

- 无权限动作不渲染。
- 不使用 disabled 灰色按钮代替。
- “新建商机”进入商机新增页并预填客户。
- “添加跟进”打开客户跟进抽屉。
- “创建拜访”进入拜访新增页并预填客户。
- 停用客户不展示新业务动作。
- 同步异常时提交新业务前必须重新校验可用性。
- More 使用 Ellipsis 图标并有 Tooltip。

## 8. 二次确认与弹窗

### 8.1 重试 ERP 同步

| 项目 | 规格 |
|------|------|
| 角色 | 系统管理员 |
| 标题 | `重试同步` |
| 内容 | `将重新拉取该客户的 ERP 最新快照，是否继续？` |
| 取消按钮 | `取消` |
| 确认按钮 | `确认重试` |
| 成功 Toast | `同步任务已重新提交` |
| 失败 Toast | `同步重试失败，请稍后再试` |

### 8.2 导出确认

- 常规导出不弹二次确认。
- 超过后台阈值时弹出异步导出提示。
- 标题：`创建导出任务`。
- 内容：`数据量较大，将生成后台导出任务。`。
- 确认按钮：`创建任务`。

### 8.3 弹窗通用规范

- 点击遮罩不关闭提交中的弹窗。
- 提交中禁用按钮。
- 服务端失败保留弹窗。
- Esc 仅在未提交时关闭。
- 焦点回到触发按钮。

## 9. 风险展示与筛选

### 9.1 行级展示

- 高风险使用强提示 Tag。
- 中风险使用警示 Tag。
- 低风险使用正常 Tag。
- 待计算显示 `待计算`。
- 计算失败且有历史结果时展示历史 Tag + 时钟图标。
- 计算失败且无历史结果时展示 `暂不可用`。

### 9.2 Tooltip

- 展示最近成功计算时间。
- 展示命中的主要条件摘要。
- 不展示模型技术参数。
- 高风险展示“建议尽快跟进”。
- ERP 信用通知结果不在列表 Tooltip 中暴露技术详情。

### 9.3 排序

- 风险列允许按后台风险优先级排序。
- 待计算和不可用排在有效风险之后。
- 排序受数据权限与查询条件约束。
- 风险刷新后保持用户当前排序。

## 10. 分页、加载与空态

### 10.1 分页

- 默认每页 20 条。
- 支持 20、50、100 条。
- 显示总条数。
- 切页滚动到表格顶部。
- 变更每页条数后回到第一页。
- URL 保存页码和每页条数。

### 10.2 加载

- 初次加载显示 10 行骨架。
- 查询刷新保留表头。
- 表格加载时不清空旧数据，覆盖半透明加载层。
- 请求返回后一次替换。
- 快速查询取消旧请求。

### 10.3 空态

| 场景 | 文案 | 动作 |
|------|------|------|
| 系统无客户 | `暂无 ERP 客户快照` | `刷新` |
| 查询无结果 | `未找到符合条件的客户` | `重置筛选` |
| 高风险为空 | `当前没有高风险客户` | 返回全部客户 |
| 同步服务异常 | `客户同步服务暂不可用` | 重试 |

空态不提供新增客户按钮。
