# 03 — 功能清单与模块地图

## 一期功能模块（5 大模块）

| 模块 | 功能点 | 优先级 |
|------|--------|--------|
| **线索管理** | 线索录入/Excel 导入、线索池、公海、分配规则（按区域/行业）、超时回收、去重校验、转客户 | P0 |
| **商机管理** | 商机创建（关联客户+商品）、阶段推进（初步接触→需求确认→报价→谈判→成交→丢单）、预计金额/预计成交日期、商机看板 | P0 |
| **客户管理** | 客户列表、客户快照（从 ERP 同步）、关联商机、关联订单、跟进记录 | P0 |
| **跟进记录** | 跟进方式（电话/拜访/邮件）、跟进内容、下次计划、关联对象（线索/商机/客户） | P0 |
| **AI 串联** | 线索评分、商机预测、流失预警 | P1 |

## 模块关系

```
线索池 → 分配 → 线索跟进 → 转客户（同步 ERP 建档）
                              ↓
                           创建商机 → 阶段推进 → 成交（ERP 下推订单）
                              ↓
                           丢单（记录原因，线索回公海）
```

## 关键规则

| 规则 | 说明 |
|------|------|
| 线索去重 | 同一手机号/邮箱不允许重复创建线索 |
| 超时回收 | 线索分配后 48 小时未跟进 → 自动回公海 |
| 客户 SSOT | 客户正式档案在 ERP，CRM 持有快照+关联数据 |
| 商机成交 | 必须关联 ERP 商品，成交时生成 ERP 销售订单草稿 |
