# 04 — 核心业务流程详解

## 线索 → 客户 主流程

```
市场渠道 → 线索录入 → 去重校验 → 线索评分(AI) → 分配销售 → 跟进
                                                      ↓
                                                  48h 未跟进 → 自动回公海
                                                      ↓
                                                  销售跟进 → 判定有效 → 转客户
                                                      ↓
                                                  判定无效 → 标记原因 → 回公海/废弃
```

## 商机 → 成交 主流程

```
客户创建商机 → 关联商品 → 需求确认 → 报价 → 谈判 → 成交/丢单
                  ↓                        ↓
            AI 商机预测(成交概率)     成交 → 下推 ERP 销售订单草稿
                  ↓                        ↓
            高概率 → 通知 ERP 预占库存   丢单 → 记录原因，商机关闭
```

## 线索公海规则

| 规则 | 说明 |
|------|------|
| 进入公海条件 | ① 分配后 48h 未跟进 ② 销售主动放弃 ③ 判定无效但保留观察 |
| 公海认领 | 任何销售可主动认领，认领后归属变更 |
| 公海保护 | 同一线索被同一销售放弃后 7 天内不可再次认领 |

## AI 评分模型输入维度

| 维度 | 权重 | 说明 |
|------|------|------|
| 线索来源 | 15% | 官网>活动>展会>批量导入 |
| 行业匹配度 | 20% | 与目标行业的契合度 |
| 跟进响应速度 | 25% | 首次响应时间越短分越高 |
| 历史转化率 | 25% | 相似画像线索的历史转化数据 |
| 最近活跃度 | 15% | 7 天内有交互加分 |

评分结果：≥80 分自动分配最优销售，50-79 进入待分配池，<50 进入培育池。
